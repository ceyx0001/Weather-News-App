Contains the algorithms of sending requests to a geolocation API and parsing the geo-coordinate data.

Get geo-coordinate API data.

Queries against the host API endpoint and returns the data that was extracted as a table represented by 2d vector. with columns separated by the data name/type. However, there is only 1 row as the request is for 1 location.
