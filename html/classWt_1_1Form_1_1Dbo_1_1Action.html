bool Wt::Form::Dbo::Action::hasDboField ( const std::string &  name)
inlineprotected

Returns whether or not the database column is used in the model.

We need to make sure the database column is actually used in the model before storing the information in the WFormModel. Otherwise WFormModel::setValue will add the column to the model, which is not what we want here.
