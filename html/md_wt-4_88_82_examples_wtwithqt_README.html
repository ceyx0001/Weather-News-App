Wt/Qt interopability example

This is an example that illustrates some aspects of using Qt code in a Wt application.

The lib/ folder contains a number of utility classes that facilitate the integration, dealing with Qt's twisted threading architecture.

How to run

See the README in the parent directory.

What it illustrates

  • the use of the wtwithqt interopability layer
