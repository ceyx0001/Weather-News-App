\doxysubsection{Detailed Description}
A C\+SS rule based on a template widget. 

This is a C\+SS rule whose C\+SS style properties are defined based on properties of a template widget. When modifying the template widget, these changes are reflected on the C\+SS rule and thus all widgets that have this C\+SS rule.
