\doxysubsection{Detailed Description}
A widget that renders a Flash object (also known as Flash movie). 

This class loads a .swf Flash file in the browser.

Flash objects must have their size set, so do not forget to call \mbox{\hyperlink{classWt_1_1WFlashObject_a1dd54a755f527db6912f0094fd177c04}{resize()}} after instantiation or your content will be invisible. Wt will modify width and height attributes of the Flash object if \mbox{\hyperlink{classWt_1_1WFlashObject_a1dd54a755f527db6912f0094fd177c04}{resize()}} is called after the object is instantiated; it is however not clear if this is permitted by the Flash plugin.

Any \mbox{\hyperlink{classWt_1_1WWidget}{W\+Widget}} can be set with \mbox{\hyperlink{classWt_1_1WFlashObject_a48449d3189fce5f0947d5e0c45ef75c5}{set\+Alternative\+Content()}}, and this widget will be shown only when the browser has no Flash support. By default, a \textquotesingle{}Download Flash\textquotesingle{} button will be displayed that links to a website where the Flash player can be downloaded. You may modify this to be any widget, such as a \mbox{\hyperlink{classWt_1_1WImage}{W\+Image}}, or a native Wt implementation of the Flash movie.

\doxyparagraph*{C\+SS}

Styling through C\+SS is not applicable.

Resizes the widget. 

Specifies a fixed size for this widget, setting C\+SS {\ttfamily width} and {\ttfamily height} properties. By default a widget has automatic width and height, which sets a size for the widget following C\+SS rules.

When the widget is not managed by a layout manager, the automatic (natural) size of a widget depends on whether they widget is a {\itshape block} or {\itshape inline} widget\+:
\begin{DoxyItemize}
\item a {\itshape block} widget takes by default the width of the parent, and the height that it needs based on its contents
\item an {\itshape inline} widget takes the width and height that it needs based on its contents (possibly wrapping over multiple lines). The width and height of an inline widget cannot be changed (by the letter of C\+SS, although most browsers will react to it in varying ways).
\end{DoxyItemize}

When inserted in a layout manager, the size set will be used as a widget\textquotesingle{}s preferred size, but the widget may be given a different size by the layout manager based on available space and stretch factors. The actual size given by a layout manager may be retrieved by making the widget \char`\"{}layout size aware\char`\"{}, using \mbox{\hyperlink{classWt_1_1WWidget_a4f83592912a7f8fa4fd35dadde78ee74}{set\+Layout\+Size\+Aware()}}. If you have defined a {\ttfamily \char`\"{}wt\+Resize()\char`\"{}} Java\+Script method for the widget, then this method will also be called.

The default width and height of a widget is \mbox{\hyperlink{classWt_1_1WLength_a0cf39ca4225776879d56ade60320c31a}{W\+Length\+::\+Auto}}.

Sets a Flash variable. 

This method is a helper function to set variable values in the flashvars parameter.

The flash variables will be properly encoded (U\+RL encoding) before being passed to the flashvars parameter.

Setting the same Flash variable a second time will overwrite the previous value. Flash variables can only be set before the widget is rendered for the first time, so it is recommended to call this method shortly after construction before returning to the idle loop.
