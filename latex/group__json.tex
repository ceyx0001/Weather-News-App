\doxysubsection{Detailed Description}
A J\+S\+ON representation and parsing library. 

The J\+S\+ON library contains data types to represent a J\+S\+ON data structure (\mbox{\hyperlink{classWt_1_1Json_1_1Value}{Value}}, \mbox{\hyperlink{classWt_1_1Json_1_1Object}{Object}} and \mbox{\hyperlink{classWt_1_1Json_1_1Array}{Array}}), a J\+S\+ON parser, and a J\+S\+ON serializer.

Usage example\+:

\DoxyCodeLine{Json::Object result;}
\DoxyCodeLine{\mbox{\hyperlink{group__json_ga439195eac280701c9a1e5f0317149866}{Json::parse}}(\textcolor{stringliteral}{"\{ "}}
\DoxyCodeLine{            \textcolor{stringliteral}{"  \(\backslash\)"a\(\backslash\)": \(\backslash\)"That's great\(\backslash\)", "}}
\DoxyCodeLine{            \textcolor{stringliteral}{"  \(\backslash\)"b\(\backslash\)": true "}}
\DoxyCodeLine{            \textcolor{stringliteral}{"\}"},}
\DoxyCodeLine{            result);}
\DoxyCodeLine{}
\DoxyCodeLine{std::cerr << \textcolor{stringliteral}{"Size: "} << result.size(); << std::endl; \textcolor{comment}{// Size: 2}}
\DoxyCodeLine{WString s = result.get(\textcolor{stringliteral}{"a"});}
\DoxyCodeLine{\textcolor{keywordtype}{bool} b = result.get(\textcolor{stringliteral}{"b"});}
