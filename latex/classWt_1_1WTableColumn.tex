\doxysubsection{Detailed Description}
A table column. 

A W\+Table\+Column is returned by \mbox{\hyperlink{classWt_1_1WTable_afc72f8bb80b6a6e659a13124a407fd98}{W\+Table\+::column\+At()}} and managing various properties of a single column in a table (it is however not a widget).

A table column corresponds to the H\+T\+ML {\ttfamily $<$col$>$} tag.
