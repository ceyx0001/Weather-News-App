If the range specification is not satisfiable, R\+FC 2616 states you should return a response status of 416. \mbox{\hyperlink{classWt_1_1Http_1_1Request_1_1ByteRangeSpecifier_a1acda3e94a0108d39eda22aea55cd2a0}{is\+Satisfiable()}} will return true if a Range header was missing or a syntax error occured, in which case the number of Byte\+Ranges will be zero and the client must send the entire file.
