\doxysubsection{Detailed Description}
A class providing details for a keyboard event. 

A key event is associated with the \mbox{\hyperlink{classWt_1_1WInteractWidget_a639e4f54902b864f211e2e6cf9f80dc7}{W\+Interact\+Widget\+::key\+Went\+Down()}}, \mbox{\hyperlink{classWt_1_1WInteractWidget_aed28d2568889f54a4809250ae265a08b}{W\+Interact\+Widget\+::key\+Went\+Up()}} and \mbox{\hyperlink{classWt_1_1WInteractWidget_a9ddbec77fbb35319948fa8436f1aa2d4}{W\+Interact\+Widget\+::key\+Pressed()}} signals.

{\footnotesize\ttfamily int Wt\+::\+W\+Key\+Event\+::char\+Code (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption}) const}



Returns the unicode character code. 

This is only defined for a \mbox{\hyperlink{classWt_1_1WInteractWidget_a9ddbec77fbb35319948fa8436f1aa2d4}{key\+Pressed }} event, and returns the unicode character code point of a character that is entered.

For the \mbox{\hyperlink{classWt_1_1WInteractWidget_a639e4f54902b864f211e2e6cf9f80dc7}{key\+Went\+Down }} and \mbox{\hyperlink{classWt_1_1WInteractWidget_aed28d2568889f54a4809250ae265a08b}{key\+Went\+Up }} events, \textquotesingle{}0\textquotesingle{} is returned.

The \mbox{\hyperlink{classWt_1_1WKeyEvent_a0685e315762ae3641c13e5e211a7af67}{char\+Code()}} may be different from \mbox{\hyperlink{classWt_1_1WKeyEvent_ad69f2c0f6ea4f4189e2db028be031254}{key()}}. For example, a \mbox{\hyperlink{}{Key\+::M}} key may correspond to \textquotesingle{}m\textquotesingle{} or \textquotesingle{}M\textquotesingle{} character, depending on whether the shift key is pressed simultaneously.
