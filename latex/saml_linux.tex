\hypertarget{saml_linux_autotoc_md23}{}\doxysubsection{From source}\label{saml_linux_autotoc_md23}
The following dependencies are usually available as a package on most distributions\+:


\begin{DoxyItemize}
\item \href{http://xerces.apache.org/xerces-c/download.cgi}{\texttt{ Xerces-\/\+C++}}
\item \href{https://santuario.apache.org/cindex.html}{\texttt{ Apache Santuario / Apache X\+ML Security for C++}}
\item \href{https://curl.se/}{\texttt{ libcurl}}
\item \href{https://www.boost.org/}{\texttt{ Boost}} (also a \mbox{\hyperlink{classWt}{Wt}} dependency)
\item \href{https://www.openssl.org/}{\texttt{ Open\+S\+SL}} (also an optional \mbox{\hyperlink{classWt}{Wt}} dependency)
\end{DoxyItemize}

You can download sources for Log4\+Shib, X\+M\+L\+Tooling, and Open\+S\+A\+ML from \href{https://shibboleth.net/downloads}{\texttt{ Shibboleth}}.


\begin{DoxyItemize}
\item \href{https://shibboleth.net/downloads/log4shib/latest/}{\texttt{ Log4\+Shib}}
\item \href{https://shibboleth.net/downloads/c++-opensaml/latest/}{\texttt{ X\+M\+L\+Tooling and Open\+S\+A\+ML}}
\end{DoxyItemize}

Installation is a simple {\ttfamily ./configure}/{\ttfamily make install}. If you\textquotesingle{}re installing into a prefix, you can use {\ttfamily P\+K\+G\+\_\+\+C\+O\+N\+F\+I\+G\+\_\+\+P\+A\+TH}\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{PKG\_CONFIG\_PATH=\$PREFIX/lib/pkgconfig ./configure -\/-\/prefix=\$PREFIX}
\DoxyCodeLine{make}
\DoxyCodeLine{make install}
\end{DoxyCode}
