\doxysubsection{Detailed Description}
Validator for password strength. 

This class defines a specialized validator interface for evaluating password strength. The implementation allows to evaluate strength in addition to the normal validator functionality of validating a password.

The \mbox{\hyperlink{classWt_1_1Auth_1_1AbstractPasswordService_1_1AbstractStrengthValidator_a930b7f74795f0a2df87212b7a1aadd06}{evaluate\+Strength()}} computes the strength and returns an instance of Strenght\+Validator\+Result which contains information on the validity and the strength of the password together with possible messages.
