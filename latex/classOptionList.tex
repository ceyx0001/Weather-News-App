\doxysubsection{Detailed Description}
A list of options, separated by \textquotesingle{}$\vert$\textquotesingle{}. 

This widget is part of the Wt composer example.

An \mbox{\hyperlink{classOptionList}{Option\+List}} displays a list of \mbox{\hyperlink{classOption}{Option}} widgets, which are separated by a \textquotesingle{}$\vert$\textquotesingle{} separator.

For example, Foo $\vert$ Bar $\vert$ Huu

When Options are hidden, the separators are adjusted so that there is no separator after the last visible option. However, this requires a call of \mbox{\hyperlink{classOptionList_a5b0013e09c5f269f33aa5c9bfb17e32b}{update()}} each time an option is hidden or shown. This is because the removing of separators is optimized in stateless implementations, and thus in client-\/side Java\+Script code. Since the behaviour is not entirely stateless, the \mbox{\hyperlink{classOptionList_a5b0013e09c5f269f33aa5c9bfb17e32b}{update()}} method resets stateless implementations if necessary.
