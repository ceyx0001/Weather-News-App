\doxysubsection{Detailed Description}
A widget that allows dropping files for upload. 

This widget accepts files that are dropped into it. A signal is triggered whenever one or more files are dropped. The filename, type and size of these files is immediately available through the \mbox{\hyperlink{classWt_1_1WFileDropWidget_1_1File}{W\+File\+Drop\+Widget\+::\+File}} interface.

The file upload is done sequentially. All files before the \mbox{\hyperlink{classWt_1_1WFileDropWidget_a3c4d94ceab18b60f7118d70ff9698596}{current\+Index()}} have either finished, failed or have been cancelled.

The widget has the default style-\/class \textquotesingle{}Wt-\/filedropzone\textquotesingle{}. The style-\/class \textquotesingle{}Wt-\/dropzone-\/hover\textquotesingle{} is added when files are hovered over the widget.
