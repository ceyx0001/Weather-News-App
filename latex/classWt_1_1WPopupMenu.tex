\doxysubsection{Detailed Description}
A menu presented in a popup window. 

The menu implements a typical context menu, with support for submenu\textquotesingle{}s. It is a specialized \mbox{\hyperlink{classWt_1_1WMenu}{W\+Menu}} from which it inherits most of the A\+PI.

When initially created, the menu is invisible, until \mbox{\hyperlink{classWt_1_1WPopupMenu_a9da2dff4c6de6845a5a15b1a7c22572c}{popup()}} or exec() is called. Then, the menu will remain visible until an item is selected, or the user cancels the menu (by hitting Escape or clicking elsewhere).

The implementation assumes availability of Java\+Script to position the menu at the current mouse position and provide feed-\/back of the currently selected item.

As with \mbox{\hyperlink{classWt_1_1WDialog}{W\+Dialog}}, there are two ways of using the menu. The simplest way is to use one of the synchronous exec() methods, which starts a reentrant event loop and waits until the user cancelled the popup menu (by hitting Escape or clicking elsewhere), or selected an item.

Alternatively, you can use one of the \mbox{\hyperlink{classWt_1_1WPopupMenu_a9da2dff4c6de6845a5a15b1a7c22572c}{popup()}} methods to show the menu and listen to the \mbox{\hyperlink{classWt_1_1WPopupMenu_a4bab7622553aaf6b050976e0f140a27f}{triggered}} signal where you read the \mbox{\hyperlink{classWt_1_1WPopupMenu_a4105a2717e2c4797043e3b9bbafd2f01}{result()}}, or associate the menu with a button using \mbox{\hyperlink{classWt_1_1WPushButton_ae919cb76993211553b50a996bdf80fe9}{W\+Push\+Button\+::set\+Menu()}}.

You have several options to react to the selection of an item\+:
\begin{DoxyItemize}
\item Either you use the \mbox{\hyperlink{classWt_1_1WMenuItem}{W\+Menu\+Item}} itself to identify the action, perhaps by specialization or simply by binding custom data using \mbox{\hyperlink{classWt_1_1WMenuItem_a4857378a84c34edc405086337b24835e}{W\+Menu\+Item\+::set\+Data()}}.
\item You can bind a separate method to each item\textquotesingle{}s \mbox{\hyperlink{classWt_1_1WMenuItem_a06ceec38d6a866c0a1305f236bd712ef}{W\+Menu\+Item\+::triggered}} signal.
\end{DoxyItemize}
