\doxysubsection{Detailed Description}
A validator for validating floating point user input. 

This validator checks whether user input is a double in the pre-\/defined range.

\doxyparagraph*{i18n}

The strings used in this class can be translated by overriding the default values for the following localization keys\+: Wt.\+W\+Double\+Validator.\+Not\+A\+Number\+: Must be a number Wt.\+W\+Double\+Validator.\+Too\+Small\+: The number must be larger than \{1\} Wt.\+W\+Double\+Validator.\+Bad\+Range\+: The number must be in the range \{1\} to \{2\} Wt.\+W\+Double\+Validator.\+Too\+Large\+: The number must be smaller than \{1\}
