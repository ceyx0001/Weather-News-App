\doxysubsection{Detailed Description}
Class representing grid-\/based data for a 3D chart. 

General information can be found at \mbox{\hyperlink{classWt_1_1Chart_1_1WAbstractDataSeries3D}{W\+Abstract\+Data\+Series3D}}. The model for this dataseries is structured as a table. One of the columns (by default index 0) contains the x-\/axis values, one of the rows (by default index 0) contains the y-\/axis values. All other values in the table contain the z-\/value corresponding to the x-\/ and y-\/values with the same column-\/ and row-\/index.
