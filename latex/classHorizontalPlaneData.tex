Returns data at a specified model index for the given role. 

You should check the {\ttfamily role} to decide what data to return. Usually a View class will ask for data for several roles which affect not only the contents (\mbox{\hyperlink{classWt_1_1ItemDataRole_ae3ecc48a21764c9fce36a9be37ccc2c1}{Wt\+::\+Item\+Data\+Role\+::\+Display}}) but also icons (\mbox{\hyperlink{classWt_1_1ItemDataRole_a9db6daf35b3c68db6df624411420d4b7}{Wt\+::\+Item\+Data\+Role\+::\+Decoration}}), U\+R\+Ls (\mbox{\hyperlink{classWt_1_1ItemDataRole_ae478217452f058add8fd204ddacde8ae}{Wt\+::\+Item\+Data\+Role\+::\+Link}}), and other visual aspects. If your item does not specify data for a particular role, it should simply return a Wt\+::cpp17\+::any().
