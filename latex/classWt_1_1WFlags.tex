\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Enum\+Type$>$\newline
class Wt\+::\+W\+Flags$<$ Enum\+Type $>$}

Utility class for type-\/safe combinations of enumeration flags. 

This type is used in Wt A\+PI whenever one or more flag options are expected, instead of an {\ttfamily int}. The class provides type safety, in the sense that it checks that the correct combination of enum values is bound to the argument, and does not cost any run-\/time overhead (internally it uses only an {\ttfamily int} to represent the combination of flags.
