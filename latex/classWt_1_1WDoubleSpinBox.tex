\doxysubsection{Detailed Description}
An input control for fixed point numbers. 

The spin box provides a control for entering a fixed point number. It consists of a line edit, and buttons which allow to increase or decrease the value. If you rather need input of an integer number number, use \mbox{\hyperlink{classWt_1_1WSpinBox}{W\+Spin\+Box}} instead.

W\+Double\+Spin\+Box is an \mbox{\hyperlink{classWt_1_1WWidget_ac78e3af143883334c82031790c87416e}{inline }} widget.

\begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{classWt_1_1WSpinBox}{W\+Spin\+Box}}
\end{DoxySeeAlso}
\begin{DoxyNote}{Note}
A spinbox configures a validator for validating the input. Therefore you cannot set a validator yourself. 
\end{DoxyNote}

A signal that indicates when the value has changed. 

This signal is emitted when \mbox{\hyperlink{classWt_1_1WFormWidget_a880589c79f1b93e7f38330e28cf6f5e2}{changed()}} is emitted, but supplies the new value as an argument. The \mbox{\hyperlink{classWt_1_1WFormWidget_a880589c79f1b93e7f38330e28cf6f5e2}{changed()}} signal is emitted when the user changes the value of the spinbox by pressing the up/down arrow, or entering a different value and pressing enter or moving focus.
