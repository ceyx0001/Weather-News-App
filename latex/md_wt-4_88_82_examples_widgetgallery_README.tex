This application illustrates a recommended deployment setup for a more involved application, separating\+:
\begin{DoxyItemize}
\item docroot\+: these are publicly available files for the browser to download through the web server (e.\+g. css, resources, ...)
\item approot\+: these are private application files opened by the application (e.\+g. resource files, config files, etc...) and not deployed in your document root
\end{DoxyItemize}

To run the application using the built-\/in httpd, you thus need to\+: \begin{DoxyVerb}cd widgetgallery/docroot
ln -s ../../../resources . # include standard Wt resource files in docroot
cd ..
../../build/examples/widgetgallery/widgetgallery.wt \
      --docroot docroot --approot approot \
      --http-address 0.0.0.0 --http-port 8080
\end{DoxyVerb}
