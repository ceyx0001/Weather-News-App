\doxysubsection{Detailed Description}
A style class for a single widget or style sheet rule. 

You can manipulate the decoration style of a single widget using \mbox{\hyperlink{classWt_1_1WWidget_ac1833c7c01599b3733712ab0bf3c3a0a}{W\+Widget\+::decoration\+Style()}} or you can use a \mbox{\hyperlink{classWt_1_1WCssDecorationStyle}{W\+Css\+Decoration\+Style}} to add a rule to the inline style sheet using \mbox{\hyperlink{classWt_1_1WCssStyleSheet_af47b055e688740b4c02f803251605a00}{W\+Css\+Style\+Sheet\+::add\+Rule(const std\+::string\&, const W\+Css\+Decoration\+Style\& style, const std\+::string\&)}}.
