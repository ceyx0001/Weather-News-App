\doxysubsection{Detailed Description}
An H\+T\+TP client message (request or response). 

This class implements a message that is sent or received by the H\+T\+TP \mbox{\hyperlink{classWt_1_1Http_1_1Client}{Client}}.

It is not to be confused with \mbox{\hyperlink{classWt_1_1Http_1_1Request}{Request}} and \mbox{\hyperlink{classWt_1_1Http_1_1Response}{Response}}, which are involved in the web application server handling.
