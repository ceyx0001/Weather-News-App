\doxysubsection{Detailed Description}
A value class that describes font metrics for a font. 

This class provides font metrics for a given font. It is returned by an implementation of \mbox{\hyperlink{classWt_1_1WPaintDevice_a468ec8684106f8e24b3c46876ee86ff7}{W\+Paint\+Device\+::font\+Metrics()}}, and may differ between devices.

All methods return pixel dimensions.
