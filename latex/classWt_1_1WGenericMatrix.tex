\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T, std\+::size\+\_\+t Rows, std\+::size\+\_\+t Cols$>$\newline
class Wt\+::\+W\+Generic\+Matrix$<$ T, Rows, Cols $>$}

A value class that describes a matrix. 

This class represents a fixed-\/size dense (!= sparse) matrix. It can be templatized to the number of rows and columns, and to the datatype stored (integer types, floatin point types, complex types, ...)

The row order of this matrix class is row-\/major. This means that when accessing the raw data store linearly, you will first encounter all elements of the first row, then the second row, and so on.

This template class is used in \mbox{\hyperlink{classWt}{Wt}} as base class for transformation matrices, but can also be used as a general matrix class. Efficiency for this use case was considered when this class was implemented, but we recommend that you use a more specialized matrix class library if the algorithms you need exceed what\textquotesingle{}s offered here (for example, if you intend to do many linear algebra computations, you may consider boost ublass, M\+TL, ...).
