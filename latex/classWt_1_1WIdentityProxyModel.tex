\doxysubsection{Detailed Description}
A proxy model that proxies its source model unmodified. 

A \mbox{\hyperlink{classWt_1_1WIdentityProxyModel}{W\+Identity\+Proxy\+Model}} simply forwards the structure of the source model, without any transformation. \mbox{\hyperlink{classWt_1_1WIdentityProxyModel}{W\+Identity\+Proxy\+Model}} can be used as a base class for implementing proxy models that reimplement \mbox{\hyperlink{classWt_1_1WAbstractProxyModel_a8e938b2a268c99bf17f0529cfffc322e}{data()}}, but retain all other characteristics of the source model.
