Sets the grouping filter(s). 

It\textquotesingle{}s like \mbox{\hyperlink{classWt_1_1Dbo_1_1AbstractQuery_a7eff29d6f7478047d8d01751692dd1a7}{where()}}, but for aggregate fields.

For example you can\textquotesingle{}t go\+:

select department.\+name, count(employees) from department where count(employees) $>$ 5 group by count(employees);

Because you can\textquotesingle{}t have aggregate fields in a where clause, but you can go\+:

select department.\+name, count(employees) from department group by count(employees) having count(employees) $>$ 5;

This will of course return all the departments with more than 5 employees (and their employee count).

\begin{DoxyNote}{Note}
You must have a group by clause, in order to have a \textquotesingle{}having\textquotesingle{} clause

This is a convenience method for creating a S\+QL query, and concatenates a new {\itshape where} condition expression to the current query.

The condition must be a valid S\+QL condition expression.

Multiple conditions may be provided by successive calls to \mbox{\hyperlink{classWt_1_1Dbo_1_1AbstractQuery_a1114844685fc80d01c2367a06f4a6110}{or\+Where()}}, and are concatenated together using \textquotesingle{}or\textquotesingle{}. Previous conditions will be surrounded by brackets and the new condition will be concatenated using \textquotesingle{}or\textquotesingle{}. For example\+:

\DoxyCodeLine{  .orWhere(\textcolor{stringliteral}{"column\_c = ?"}).bind(\textcolor{stringliteral}{"C"});}
\end{DoxyCode}

Adds a query condition. 

This is a convenience method for creating a S\+QL query, and concatenates a new {\itshape where} condition expression to the current query.

The condition must be a valid S\+QL condition expression.

Multiple conditions may be provided by successive calls to \mbox{\hyperlink{classWt_1_1Dbo_1_1AbstractQuery_a7eff29d6f7478047d8d01751692dd1a7}{where()}}, which must each be fulfilled, and are concatenated together using \textquotesingle{}and\textquotesingle{}.

As with any part of the S\+QL query, a condition may contain positional markers \textquotesingle{}?\textquotesingle{} to which values may be bound using \mbox{\hyperlink{classWt_1_1Dbo_1_1AbstractQuery_aaaa2d59ac4afee71920e336e41440128}{bind()}}.
