\doxysubsection{Detailed Description}
A Firebird connection. 

This class provides the backend implementation for Firebird databases. It supports Firebird databases with version 2.\+1 or higher.

\begin{DoxyWarning}{Warning}
The Firebird backend is {\bfseries{U\+N\+M\+A\+I\+N\+T\+A\+I\+N\+ED}}, and may not work at all with recent versions of Firebird. This backend is based on I\+B\+PP, a project which has not seen any activity in a very long time. Emweb has no plans to continue supporting the Firebird backend. 
\end{DoxyWarning}
