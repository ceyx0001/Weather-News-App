\doxysubsection{Detailed Description}
A horizontal or vertical slider control. 

A slider allows the user to specify an integer value within a particular range using a visual slider.

The slider must be sized explicitly using \mbox{\hyperlink{classWt_1_1WSlider_a2e87d0679b0fcc17216ac7eccf5383aa}{resize()}} or by a layout manager. The default size is 150 x 50 pixels for a horizontal slider, and 50 x 150 pixels for a vertical slider.



\doxyparagraph*{C\+SS}

The non-\/native slider (H\+T\+M\+L4, see \mbox{\hyperlink{classWt_1_1WSlider_ac5a645c0b1ddd14ffc68a317f4d7ac53}{set\+Native\+Control()}}) is styled by the current C\+SS theme.

Specifies a fixed size for this widget, setting C\+SS {\ttfamily width} and {\ttfamily height} properties. By default a widget has automatic width and height, which sets a size for the widget following C\+SS rules.

When the widget is not managed by a layout manager, the automatic (natural) size of a widget depends on whether they widget is a {\itshape block} or {\itshape inline} widget\+:
\begin{DoxyItemize}
\item a {\itshape block} widget takes by default the width of the parent, and the height that it needs based on its contents
\item an {\itshape inline} widget takes the width and height that it needs based on its contents (possibly wrapping over multiple lines). The width and height of an inline widget cannot be changed (by the letter of C\+SS, although most browsers will react to it in varying ways).
\end{DoxyItemize}

When inserted in a layout manager, the size set will be used as a widget\textquotesingle{}s preferred size, but the widget may be given a different size by the layout manager based on available space and stretch factors.
