\mbox{\hyperlink{classWt}{Wt}} is a C++ library for developing web applications. It consists of\+:


\begin{DoxyItemize}
\item libwt, a widget/rendering library
\item libwthttp, an (async I/O) H\+T\+T\+P/\+Web\+Sockets server
\item libwtfcgi, a Fast\+C\+GI connector library (Unix)
\item libwtisapi, an I\+S\+A\+PI connector library (Windows)
\item libwttest, a test connector environment
\end{DoxyItemize}

It also contains a C++ O\+RM, which can be used in a web application (obviously), but can also be used on its own\+:


\begin{DoxyItemize}
\item libwtdbo, a C++ O\+RM
\item libwtdbopostgres, Postgre\+S\+QL backend
\item libwtdbosqlite3, Sqlite3 backend
\item libwtdbomysql, My\+S\+QL and Maria\+DB backend
\item libwtdbomssqlserver, Microsoft S\+QL Server backend
\item libwtdbofirebird, Firebird backend
\end{DoxyItemize}

For more information, see \href{http://www.webtoolkit.eu/wt}{\texttt{ the homepage}}.\hypertarget{md_wt-4_88_82_README_autotoc_md138}{}\doxysection{Dependencies}\label{md_wt-4_88_82_README_autotoc_md138}
To build \mbox{\hyperlink{classWt}{Wt}} from source you will need at least \href{https://cmake.org/}{\texttt{ C\+Make}}, \href{http://www.boost.org}{\texttt{ boost}}, and a C++ compiler like G\+CC, Clang or M\+S\+VC. See the minimum dependency versions for an explanation about which versions \mbox{\hyperlink{classWt}{Wt}} supports.

Optionally, you may want to add\+:


\begin{DoxyItemize}
\item \href{https://www.openssl.org}{\texttt{ Open\+S\+SL}} for S\+SL and Web\+Sockets support in the built-\/in httpd, the H\+T\+T\+P(\+S) client, and additional cryptographic hashes in the authentication module
\item \href{http://libharu.org}{\texttt{ Haru P\+DF library}} which is used for painting to P\+DF
\item \href{http://www.graphicsmagick.org/}{\texttt{ Graphics\+Magick}} which is used for painting to P\+NG, G\+IF (on Windows, Direct2D can be used instead)
\item \href{https://www.postgresql.org/}{\texttt{ Postgre\+S\+QL}} for the Postgre\+S\+QL Dbo backend
\item \href{https://www.mysql.com}{\texttt{ My\+S\+QL}} or \href{https://mariadb.org/}{\texttt{ Maria\+DB}} for the My\+S\+QL Dbo backend
\item An \href{https://docs.microsoft.com/en-us/sql/connect/odbc/download-odbc-driver-for-sql-server}{\texttt{ O\+D\+BC driver}} for the Microsoft S\+QL Server Dbo backend, and \href{http://www.unixodbc.org/}{\texttt{ unix\+O\+D\+BC}} on Unix-\/like platforms
\item \href{http://www.firebirdsql.org/}{\texttt{ Firebird}} for the Firebird Dbo backend
\item \href{http://www.pango.org/}{\texttt{ Pango}} for improved font support in P\+DF and raster image painting. On Windows, Direct\+Write can be used instead.
\item \href{https://zlib.net/}{\texttt{ Z\+Lib}} for compression in the built-\/in httpd.
\end{DoxyItemize}
