\doxysubsection{Detailed Description}
A client-\/side Java\+Script vector. 

Using a \mbox{\hyperlink{classWt_1_1WGLWidget_1_1JavaScriptVector}{Java\+Script\+Vector}}, GL parameters can be modified without communication with the server. The value of the \mbox{\hyperlink{classWt_1_1WGLWidget_1_1JavaScriptMatrix4x4}{Java\+Script\+Matrix4x4}} is updated server-\/side whenever an event is sent to the server.

The \mbox{\hyperlink{classWt_1_1WGLWidget_1_1JavaScriptVector}{Java\+Script\+Vector}} is represented in Java\+Script as an array, either as a Float32\+Array or as a plain Java\+Script array.
