\doxysubsection{Detailed Description}
A class for grouping radio buttons logically together. 

A button group manages a set of \mbox{\hyperlink{classWt_1_1WRadioButton}{radio buttons}}, making them exclusive of each other.

It is not a widget, but instead provides only a logical grouping. Radio buttons are aware of the group in which they have been added, see \mbox{\hyperlink{classWt_1_1WRadioButton_a0de845c1493cc1c9aed7b058cde7fcb1}{W\+Radio\+Button\+::group()}}. When a button is deleted, it is automatically removed its button group.

It allows you to associate id\textquotesingle{}s with each button, which you may use to identify a particular button. The special value of -\/1 is reserved to indicate {\itshape no button}.
