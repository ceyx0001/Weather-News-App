\doxysubsection{Detailed Description}
O\+Auth 2.\+0 client implementing Open\+ID Connect, a.\+k.\+a. relying party.

This class represents a client. It is a value class that stores only the id and a reference to an \mbox{\hyperlink{classWt_1_1Auth_1_1AbstractUserDatabase}{Abstract\+User\+Database}} to access its properties.

An object can point to a valid client, or be invalid. Invalid clients are typically used as return value for database queries which did not match with an existing client.
