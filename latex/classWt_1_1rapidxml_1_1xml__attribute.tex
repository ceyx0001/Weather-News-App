class Wt\+::rapidxml\+::xml\+\_\+attribute$<$ Ch $>$}

Class representing attribute node of X\+ML document. Each attribute has name and value strings, which are available through name() and value() functions (inherited from \mbox{\hyperlink{classWt_1_1rapidxml_1_1xml__base}{xml\+\_\+base}}). \mbox{\hyperlink{classNote}{Note}} that after parse, both name and value of attribute will point to interior of source text used for parsing. Thus, this text must persist in memory for the lifetime of attribute.
