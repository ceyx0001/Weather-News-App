\doxysubsection{Detailed Description}
A table row. 

A W\+Table\+Row is returned by \mbox{\hyperlink{classWt_1_1WTable_ad773593b3a5521d385497d54cd651bdf}{W\+Table\+::row\+At()}} and managing various properties of a single row in a table (it is however not a widget).

A table row corresponds to the H\+T\+ML {\ttfamily $<$tr$>$} tag.
