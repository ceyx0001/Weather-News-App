\doxysubsection{Detailed Description}
Class representing grid-\/based data for on a 3D chart. 

General information can be found at \mbox{\hyperlink{classWt_1_1Chart_1_1WAbstractDataSeries3D}{W\+Abstract\+Data\+Series3D}}. Information on how the model is structured is provided in the subclasses. Grid\+Data can be represented in three ways. This is indicated by \mbox{\hyperlink{group__charts_gacca4e123c9e4b212050bcfac488fceee}{Series3\+D\+Type}} and can be either \mbox{\hyperlink{group__charts_ggacca4e123c9e4b212050bcfac488fceeea2a3cd5946cfd317eb99c3d32e35e2d4c}{Series3\+D\+Type\+::\+Point}}, \mbox{\hyperlink{group__charts_ggacca4e123c9e4b212050bcfac488fceeeaaa0d528ba11ea1485d466dfe1ea40819}{Series3\+D\+Type\+::\+Surface}} or \mbox{\hyperlink{group__charts_ggacca4e123c9e4b212050bcfac488fceeeaddc35f88fa71b6ef142ae61f35364653}{Series3\+D\+Type\+::\+Bar}}. \mbox{\hyperlink{classNote}{Note}} that points and surfaces can only be added to charts of type \mbox{\hyperlink{group__charts_gga8d63464f873580c77508e1c0c26cbfeaa09870720ca8134284e4e305ac6ce5f19}{Chart\+Type\+::\+Scatter}}, while bars can only be added to charts of type \mbox{\hyperlink{group__charts_gga8d63464f873580c77508e1c0c26cbfeaa3adbdb3ac060038aa0e6e6c138ef9873}{Chart\+Type\+::\+Category}}.

When the data is shown as a surface, a mesh can be added to the surface. This draws lines over the surface at the positions of the x-\/ and y-\/values. For bar-\/series data, it is possible to adjust the width of the bars in both directions.

The three types of data-\/representation are illustrated below.
