\begin{DoxyItemize}
\item Debian oldstable
\item The previous L\+TS version of Ubuntu
\item The previous release of Red Hat Enterprise Linux (and its derivatives)
\item The previous release of Free\+B\+SD
\item What was the latest version of Visual Studio 5 years ago
