\doxysubsection{Detailed Description}
An abstract widget that can receive user-\/interface interaction. 

This abstract widget provides access to event signals that correspond to user-\/interface interaction through mouse or keyboard.

When Java\+Script is disabled, only the \mbox{\hyperlink{classWt_1_1WInteractWidget_af6cb297f8593e6884de66d23ea69d49c}{clicked()}} event will propagate (but without event details information).

\doxyparagraph*{C\+SS}

Configure dragging for drag and drop. 

Enable drag\&drop for this widget. The mime\+Type is used to find a suitable drop target, which must accept dropping of this mimetype.

By default, the entire widget is dragged. One may specify another widget to be dragged (for example the parent as {\ttfamily drag\+Widget}) or a {\ttfamily drag\+Widget} whose function is only to represent the drag visually (when {\ttfamily is\+Drag\+Widget\+Only} = {\ttfamily true}).

The widget to be identified as source in the drop\+Event may be given explicitly, and will default to this widget otherwise.

When using a touch interface, the widget can also be dragged after a long press.

\begin{DoxyNote}{Note}
When Java\+Script is disabled, drag\&drop does not work.
