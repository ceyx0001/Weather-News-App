\doxysubsection{Detailed Description}
An abstract base class for layout managers. 

This class is the abstract base class for any layout manager. A layout manager is associated with a container widget, and manages the layout of children inside the whole space available to the container widget.

The implementation of the layout manager depends on the container widget to which it is set, and is therefore deferred to \mbox{\hyperlink{classWt_1_1WLayoutImpl}{W\+Layout\+Impl}}.

\begin{DoxyNote}{Note}
When applying a layout manager to a \mbox{\hyperlink{classWt_1_1WContainerWidget}{W\+Container\+Widget}}, you may not define any padding for the container widget. Instead, use \mbox{\hyperlink{classWt_1_1WLayout_af18bae8c743c9f9a5cabee58826ffb45}{set\+Contents\+Margins()}}. 
\end{DoxyNote}
