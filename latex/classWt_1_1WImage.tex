\doxysubsection{Detailed Description}
A widget that displays an image. 

The image may be specified either as a U\+RL, or may be dynamically generated by a \mbox{\hyperlink{classWt_1_1WResource}{W\+Resource}}.

You may listen to events by attaching event listeners to signals such as \mbox{\hyperlink{classWt_1_1WInteractWidget_af6cb297f8593e6884de66d23ea69d49c}{clicked()}}. Since mouse events pass the coordinates through a \mbox{\hyperlink{classWt_1_1WMouseEvent}{W\+Mouse\+Event}} object, it is possible to react to clicks in specific parts of the image. An alternative is to define interactive areas on the image using \mbox{\hyperlink{classWt_1_1WImage_a11305acf84fc3c3bf69ae8812f29649d}{add\+Area()}}, which in addition allows to have customized tool tips for certain image areas (using \mbox{\hyperlink{classWt_1_1WAbstractArea_a310e8e1d6360d22bf8bf0e484a52b8f9}{W\+Abstract\+Area\+::set\+Tool\+Tip()}}).

W\+Image is an \mbox{\hyperlink{classWt_1_1WWidget_ac78e3af143883334c82031790c87416e}{inline }} widget.

\doxyparagraph*{C\+SS}

The widget corresponds to the H\+T\+ML {\ttfamily $<$img$>$} tag and does not provide styling. It can be styled using inline or external C\+SS as appropriate.

Adds an interactive area. 

Adds the {\ttfamily area} which listens to events in a specific region of the image. Areas are organized in an indexed list, to which the given {\ttfamily area} is appended. When areas overlap, the area with the lowest index receives the event.

Ownership of the {\ttfamily area} is transferred to the image.

Inserts an interactive area. 

Inserts the {\ttfamily area} which listens to events in the coresponding area of the image. Areas are organized in a list, and the {\itshape area} is inserted at index {\ttfamily index}. When areas overlap, the area with the lowest index receives the event.

Ownership of the {\ttfamily area} is transferred to the image.

Sets an alternate text. 

The alternate text should provide a fallback for browsers that do not display an image. If no sensible fallback text can be provided, an empty text is preferred over nonsense.

This should not be confused with \mbox{\hyperlink{classWt_1_1WWebWidget_a21e46c6f9797978981e437c5eeccc77f}{tool\+Tip()}} text, which provides additional information that is displayed when the mouse hovers over the image.

The default alternate text is an empty text (\char`\"{}\char`\"{}).
