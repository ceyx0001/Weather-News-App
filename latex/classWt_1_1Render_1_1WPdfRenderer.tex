\doxysubsection{Detailed Description}
An X\+H\+T\+ML to P\+DF renderer. 

This class implements an X\+H\+T\+ML to P\+DF renderer. The rendering engine supports only a subset of X\+H\+T\+ML. See the documentation of \mbox{\hyperlink{classWt_1_1Render_1_1WTextRenderer}{W\+Text\+Renderer}} for more information.

By default it uses a pixel resolution of 72 D\+PI, which is the default for a \mbox{\hyperlink{classWt_1_1WPdfImage}{W\+Pdf\+Image}}, but differs from the default used by most browsers (which is 96 D\+PI and has nothing to do with the actual screen resolution). The pixel resolution can be configured using \mbox{\hyperlink{classWt_1_1Render_1_1WPdfRenderer_aca6980c07406506c5c59b51ad4d3f555}{set\+Dpi()}}. Increasing the resolution has the effect of scaling down the rendering. This can be used in conjunction with \mbox{\hyperlink{classWt_1_1Render_1_1WTextRenderer_a2ca18a73ebbe013b6913a277408eb2a6}{set\+Font\+Scale()}} to scale the font size differently than other content.

Font information is embedded in the P\+DF. Fonts supported are native Post\+Script fonts (Base-\/14) (only A\+S\+C\+I\+I-\/7), or true type fonts (Unicode). See \mbox{\hyperlink{classWt_1_1Render_1_1WPdfRenderer_abc2ad902bd0575905297db8479ad51b6}{add\+Font\+Collection()}} for more information on how fonts are located.

Sets the resolution. 

The resolution used between C\+SS pixels and actual page dimensions. \mbox{\hyperlink{classNote}{Note}} that his does not have an effect on the {\itshape de facto} standard C\+SS resolution of 96 D\+PI that is used to convert between physical \mbox{\hyperlink{classWt_1_1WLength}{W\+Length}} units (like {\itshape cm}, {\itshape inch} and {\itshape point}) and pixels. Instead it has the effect of scaling down or up the rendered X\+H\+T\+ML on the page.

The dpi setting also affects the \mbox{\hyperlink{classWt_1_1Render_1_1WPdfRenderer_a6e7e64ed692a8cee627308fb427623dd}{page\+Width()}}, \mbox{\hyperlink{classWt_1_1Render_1_1WPdfRenderer_ac206f10a47692911b35fdca090e67220}{page\+Height()}}, and \mbox{\hyperlink{classWt_1_1Render_1_1WPdfRenderer_a4e702ff3d62d3d6b1d5d28540cede72d}{margin()}} pixel calculations.
