\doxysubsection{Detailed Description}
A validator that checks user input against a regular expression. 

This validator checks whether user input matches the given regular expression. It checks the complete input; prefix $^\wedge$ and suffix \$ are not needed.

The regex should be specified using E\+C\+M\+A\+Script syntax (\href{http://en.cppreference.com/w/cpp/regex/ecmascript}{\texttt{ http\+://en.\+cppreference.\+com/w/cpp/regex/ecmascript}})

\begin{DoxyNote}{Note}
This validator does not fully support unicode\+: it matches on the Char\+Encoding\+::\+U\+T\+F8-\/encoded representation of the string.
\end{DoxyNote}
