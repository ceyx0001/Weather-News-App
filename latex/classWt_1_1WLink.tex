\doxysubsection{Detailed Description}
A value class that defines a hyperlink target. 

This class abstracts a link target. Depending on the context, it may reference a U\+RL, a dynamic \mbox{\hyperlink{classWt_1_1WResource}{resource}}, or (for certain usages) an internal path.

Sets the location where the linked content should be displayed. 

By default, the linked content is displayed in the application (Wt\+::\+Link\+Target\+::\+Self). When the destination is an H\+T\+ML document, the application is replaced with the new document. When the link is to a document that cannot be displayed in the browser, it is offered for download or opened using an external program, depending on browser settings.

By setting {\ttfamily target} to Wt\+::\+Link\+Target\+::\+New\+Window, the destination is displayed in a new browser window or tab.
