\doxysubsection{Detailed Description}
An abstract interface for a loading indicator. 

The loading indicator displays a message while a response from the server is pending.

The widget will be shown and hidden using \mbox{\hyperlink{classWt_1_1WWidget_a52dcef5a385ddfa0a8c3e6c20000f181}{W\+Widget\+::show()}} and \mbox{\hyperlink{classWt_1_1WWidget_a0825c3ccbd4999afc1a88fafa6aa6fc7}{W\+Widget\+::hide()}}. If you want to customize this behaviour, you should reimplement the \mbox{\hyperlink{classWt_1_1WWidget_a66bdda41220267a8c7ff9865996e2675}{W\+Widget\+::set\+Hidden()}} method.
