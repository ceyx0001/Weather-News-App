\doxysubsection{Detailed Description}
A value class that defines a 2D affine transformation matrix. 

The matrix is encoded using 6 parameters\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{classWt_1_1WTransform_ae1166875478011bbaa47b0c143c8650e}{m11}}  \mbox{\hyperlink{classWt_1_1WTransform_a2f28772a3535c810d8af5421777b4a6f}{m12}}   0}
\DoxyCodeLine{\mbox{\hyperlink{classWt_1_1WTransform_a10ce670920d570d07e91d1dd846d02e4}{m21}}  \mbox{\hyperlink{classWt_1_1WTransform_a21d7e4b130fc196923ce3a1c3b5f9a62}{m22}}   0}
\DoxyCodeLine{\mbox{\hyperlink{classWt_1_1WTransform_a0acf97ec22b09d77fb18ec363bdfafb7}{dx}}   \mbox{\hyperlink{classWt_1_1WTransform_ab970a37ccdd388af73a9d45e44b63e36}{dy}}    1}
\end{DoxyCode}


In this representation, \mbox{\hyperlink{classWt_1_1WTransform_a0acf97ec22b09d77fb18ec363bdfafb7}{dx()}} (= \mbox{\hyperlink{classWt_1_1WTransform_a1b26e47e25f8b2254fad617dd8d2f17e}{m31()}}) and \mbox{\hyperlink{classWt_1_1WTransform_ab970a37ccdd388af73a9d45e44b63e36}{dy()}} (= \mbox{\hyperlink{classWt_1_1WTransform_a27c83995085447e1de0e179b589db1df}{m32()}}) represent the translation components, and m{\itshape xy} represent a 2D matrix that contains the scale, rotation (and skew) components.

The transformation is used to represent a tansformed coordinate system, and provides methods to \mbox{\hyperlink{classWt_1_1WTransform_a9fb70c530d57c5b3826c3266b7bd1cbd}{rotate()}}, \mbox{\hyperlink{classWt_1_1WTransform_a2bffeef816ccb2be161fdc7c2d339e5d}{scale()}}, \mbox{\hyperlink{classWt_1_1WTransform_a58dc3da9a195adc212e9001faaa152bf}{shear()}} or \mbox{\hyperlink{classWt_1_1WTransform_a5819b2d1b3681abeca43d2fc05b4732a}{translate()}} this coordinate system.

There are also 2 methods to decompose an arbitrary matrix into elementary operations\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{classWt_1_1WTransform_a6949998bdf1a6fa12646d8fd6c6c056a}{decompose\+Translate\+Rotate\+Scale\+Skew()}} decomposes into a {\itshape T} \&\#x2218; {\itshape R} \&\#x2218; {\itshape Sxx} \&\#x2218; {\itshape Sxy}
\item \mbox{\hyperlink{classWt_1_1WTransform_ab0f477fb4c6a9102638d72be27665b37}{decompose\+Translate\+Rotate\+Scale\+Rotate()}} decomposes into a {\itshape T} \&\#x2218; {\itshape R1} \&\#x2218; {\itshape Sxx} \&\#x2218; {\itshape R2}
\end{DoxyItemize}

with {\itshape T} a translation, {\itshape R} a rotation, {\itshape Sxx} a scale, and {\itshape Sxy} a skew component.

\doxyparagraph*{Java\+Script exposability}

A W\+Transform is Java\+Script exposable. If a W\+Transform \mbox{\hyperlink{classWt_1_1WJavaScriptExposableObject_a4e121ccbb14715c33dc1b019b6636156}{is Java\+Script bound}}, it can be accessed in your custom Java\+Script code through \mbox{\hyperlink{classWt_1_1WJavaScriptHandle_a0d0bc00f0bcafb596c3f6c96c4241ded}{its handle\textquotesingle{}s js\+Ref()}}. A transform is represented as a Java\+Script array, e.\+g. a W\+Transform(m11, m12, m21, m22, dx, dy) will be represented in Java\+Script by\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{[\mbox{\hyperlink{classWt_1_1WTransform_ae1166875478011bbaa47b0c143c8650e}{m11}}, \mbox{\hyperlink{classWt_1_1WTransform_a2f28772a3535c810d8af5421777b4a6f}{m12}}, \mbox{\hyperlink{classWt_1_1WTransform_a10ce670920d570d07e91d1dd846d02e4}{m21}}, \mbox{\hyperlink{classWt_1_1WTransform_a21d7e4b130fc196923ce3a1c3b5f9a62}{m22}}, \mbox{\hyperlink{classWt_1_1WTransform_a0acf97ec22b09d77fb18ec363bdfafb7}{dx}}, \mbox{\hyperlink{classWt_1_1WTransform_ab970a37ccdd388af73a9d45e44b63e36}{dy}}]}
\end{DoxyCode}


As an exception to the general rule that \mbox{\hyperlink{classWt_1_1WJavaScriptExposableObject_a4e121ccbb14715c33dc1b019b6636156}{Java\+Script bound}} objects should not be modified, W\+Transform does support many modifications.
