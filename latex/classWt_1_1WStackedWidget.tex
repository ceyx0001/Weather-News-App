\doxysubsection{Detailed Description}
A container widget that stacks its widgets on top of each other. 

This is a container widget which at all times has only one item visible. The widget accomplishes this using set\+Hidden(bool) on the children.

Using \mbox{\hyperlink{classWt_1_1WStackedWidget_a3192b31ecdc24848cbbe9173d899c66a}{current\+Index()}} and \mbox{\hyperlink{classWt_1_1WStackedWidget_a4a2f79cf685ade6ad529e0673e22e6b0}{set\+Current\+Index(int index)}} you can retrieve or set the visible widget.

W\+Stacked\+Widget, like \mbox{\hyperlink{classWt_1_1WContainerWidget}{W\+Container\+Widget}}, is by default not inline.

\doxyparagraph*{C\+SS}

The widget is rendered using an H\+T\+ML {\ttfamily $<$div$>$} tag and does not provide styling. It can be styled using inline or external C\+SS as appropriate.

The animation is used to hide the previously current widget and show the next current widget using \mbox{\hyperlink{classWt_1_1WStackedWidget_a4a2f79cf685ade6ad529e0673e22e6b0}{set\+Current\+Index()}}.

The initial value for {\ttfamily animation} is W\+Animation(), specifying no animation.

When {\ttfamily auto\+Reverse} is set to {\ttfamily true}, then the reverse animation is chosen when the new index precedes the current index. This only applies to Animation\+Effect\+::\+Slide\+In\+From\+Left, Animation\+Effect\+::\+Slide\+In\+From\+Right, Animation\+Effect\+::\+Slide\+In\+From\+Top or Animation\+Effect\+::\+Slide\+In\+From\+Bottom transition effects.

\begin{DoxyNote}{Note}
If you intend to use a transition animation with a \mbox{\hyperlink{classWt_1_1WStackedWidget}{W\+Stacked\+Widget}} you should set it before it is first rendered. Otherwise, transition animations caused by \mbox{\hyperlink{classWt_1_1WStackedWidget_a4a2f79cf685ade6ad529e0673e22e6b0}{set\+Current\+Index()}} may not be correctly performed. If you do want to force this change you can use \mbox{\hyperlink{classWt_1_1WApplication_a48ceb7972ef7ec0e557adf441f660f2f}{W\+Application\+::process\+Events}} before calling \mbox{\hyperlink{classWt_1_1WStackedWidget_a4a2f79cf685ade6ad529e0673e22e6b0}{set\+Current\+Index()}}.

It is also not supported to use a Animation\+Effect\+::\+Pop animation on a \mbox{\hyperlink{classWt_1_1WStackedWidget}{W\+Stacked\+Widget}}.
\end{DoxyNote}
