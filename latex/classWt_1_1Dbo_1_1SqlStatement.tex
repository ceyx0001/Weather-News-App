\doxysubsection{Detailed Description}
Abstract base class for a prepared S\+QL statement. 

The statement may be used multiple times, but cannot be used concurrently. It also cannot be copied.

This class is part of \mbox{\hyperlink{namespaceWt_1_1Dbo}{Wt\+::\+Dbo}}\textquotesingle{}s backend A\+PI, and should not be used directly. Its interface must be reimplemented for each backend corresponding to a supported database.
