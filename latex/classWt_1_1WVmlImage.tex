\doxysubsection{Detailed Description}
A paint device for rendering using the V\+ML pseudo-\/standard. 

The W\+Vml\+Image is used by \mbox{\hyperlink{classWt_1_1WPaintedWidget}{W\+Painted\+Widget}} to render to the browser using the Vector Markup Language (V\+ML) (to support graphics on Internet Explorer browsers).

\begin{DoxyNote}{Note}
The current implementation has only limited support for clipping\+: only rectangular areas aligned with the X/Y axes can be used as clipping path. 
\end{DoxyNote}
