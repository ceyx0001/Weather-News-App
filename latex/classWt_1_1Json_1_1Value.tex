\doxysubsection{Detailed Description}
A J\+S\+ON value. 

This class represents a J\+S\+ON value, which may be\+:
\begin{DoxyItemize}
\item a simple type (boolean, number, string)
\item a composed type (\mbox{\hyperlink{classWt_1_1Json_1_1Array}{Array}} or \mbox{\hyperlink{classWt_1_1Json_1_1Object}{Object}})
\item \mbox{\hyperlink{classWt_1_1Json_1_1Value_a344e20a378d9e91effbf91772b409942}{Null}}\+: this represents values which are represented as \char`\"{}null\char`\"{} in J\+S\+ON notations, but also the values returned for missing members in an \mbox{\hyperlink{classWt_1_1Json_1_1Object}{Object}}, or values which are the result of a failed type coercion. 
\end{DoxyItemize}
