\subsubsection*{template$<$class C$>$\newline
class Wt\+::\+Form\+::\+Dbo\+::\+Form\+View$<$ C $>$}

A view class to represent database objects. 

This class takes the database object as a template argument. Instead of having to manually define all the widgets, the system will automatically generate default widgets based on the data type used in the database. For example\+: a {\ttfamily \mbox{\hyperlink{classWt_1_1WString}{W\+String}}} object will by default be represented by a {\ttfamily \mbox{\hyperlink{classWt_1_1WLineEdit}{W\+Line\+Edit}}} in the UI.
