\doxysubsection{Detailed Description}
A nested class of \mbox{\hyperlink{classWt_1_1WFileDropWidget}{W\+File\+Drop\+Widget}} representing a file. 

The methods returning the filename, mime-\/type and size return valid values if the upload of this file is not yet finished. The method \mbox{\hyperlink{classWt_1_1WFileDropWidget_1_1File_a5fd232744d8e225d5e80b12ca0383a89}{uploaded\+File()}} is only available after the upload is finished.
