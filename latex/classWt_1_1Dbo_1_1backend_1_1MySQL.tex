\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classWt_1_1Dbo_1_1backend_1_1MySQL}{My\+S\+QL}} connection. 

This class provides the backend implementation for mariadb databases. It has been tested against \mbox{\hyperlink{classWt_1_1Dbo_1_1backend_1_1MySQL}{My\+S\+QL}} 5.\+6.

In order to work properly with \mbox{\hyperlink{namespaceWt_1_1Dbo}{Wt\+::\+Dbo}}, \mbox{\hyperlink{classWt_1_1Dbo_1_1backend_1_1MySQL}{My\+S\+QL}} must be configured with Inno\+DB (for \mbox{\hyperlink{classWt_1_1Dbo_1_1backend_1_1MySQL}{My\+S\+QL}}) or Xtra\+DB (for mariadb) as the default database engine -\/ so that the transaction based functionality works.

\begin{DoxyNote}{Note}
There is a bug in the implementation of milliseconds in mariadb C client which affects \mbox{\hyperlink{classWt_1_1WTime}{W\+Time}} and posix\+::time\+\_\+duration values -- it goes berserk when fractional part = 0. 
\end{DoxyNote}
