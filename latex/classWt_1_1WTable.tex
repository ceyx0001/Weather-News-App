\doxysubsection{Detailed Description}
A container widget which provides layout of children in a table grid. 

A W\+Table arranges its children in a table.

To insert or access contents, use \mbox{\hyperlink{classWt_1_1WTable_abb23850096316e37d2b77dced200514a}{element\+At(int row, int column)}} to access the \mbox{\hyperlink{classWt_1_1WTableCell}{cell}} at a particular location in the table. The W\+Table expands automatically to create the indexed (row, column) as necessary.

It is possible to insert and delete entire rows or columns from the table using the insert\+Column(int column), insert\+Row(int row), delete\+Column(int column), or delete\+Row(int row) methods.

You may indicate a number of rows and columns that act as headers using \mbox{\hyperlink{classWt_1_1WTable_aff5f29161a03a71b162f37e4e92a25c5}{set\+Header\+Count()}}. Header cells are rendered as {\ttfamily $<$th$>$} instead of {\ttfamily $<$td$>$} elements. By default, no rows or columns are configured as headers.

W\+Table is displayed as a \mbox{\hyperlink{classWt_1_1WWidget_ac78e3af143883334c82031790c87416e}{block}}.

\doxyparagraph*{C\+SS}

The widget corresponds to the H\+T\+ML {\ttfamily $<$table$>$} tag and does not provide styling. It can be styled using inline or external C\+SS as appropriate.
