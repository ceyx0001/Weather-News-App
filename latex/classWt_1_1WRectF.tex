\doxysubsection{Detailed Description}
A value class that defines a rectangle. 

The rectangle is defined by a top-\/left point and a width and height.

\doxyparagraph*{Java\+Script exposability}

A W\+RectF is Java\+Script exposable. If a W\+RectF \mbox{\hyperlink{classWt_1_1WJavaScriptExposableObject_a4e121ccbb14715c33dc1b019b6636156}{is Java\+Script bound}}, it can be accessed in your custom Java\+Script code through \mbox{\hyperlink{classWt_1_1WJavaScriptHandle_a0d0bc00f0bcafb596c3f6c96c4241ded}{its handle\textquotesingle{}s js\+Ref()}}. A rectangle is represented in Java\+Script as an array of four elements (x,y,width,height), e.\+g. a rectangle W\+Rect\+F(10,20,30,40) will be represented in Java\+Script as\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{[10, 20, 30, 40]}
\end{DoxyCode}


\begin{DoxyWarning}{Warning}
A W\+RectF that is Java\+Script exposed should be modified only through its \mbox{\hyperlink{classWt_1_1WJavaScriptHandle}{handle}}. Any attempt at modifying it will cause an exception to be thrown.
\end{DoxyWarning}
