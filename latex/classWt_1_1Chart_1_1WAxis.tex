\doxysubsection{Detailed Description}
Class which represents an axis of a cartesian chart. 

A cartesian chart has two or three axes\+: an X axis (\mbox{\hyperlink{group__charts_ggaddb678f67fbd9f374fcee0922f158c9fa02129bb861061d1a052c592e2dc6b383}{Axis\+::X}}), a Y axis (\mbox{\hyperlink{group__charts_ggaddb678f67fbd9f374fcee0922f158c9fa57cec4137b614c87cb4e24a3d003a3e0}{Axis\+::Y}}) and optionally a second Y axis (\mbox{\hyperlink{group__charts_ggaddb678f67fbd9f374fcee0922f158c9fa2aa016bd54aa28710a128dff5a42e456}{Axis\+::\+Y2}}). Each of the up to three axes in a cartesian chart has a unique \mbox{\hyperlink{classWt_1_1Chart_1_1WAxis_a3cd1146e8ebffe7eac9ea725f56fa4c0}{id()}} that identifies which of these three axes it is in the enclosing chart().

Use \mbox{\hyperlink{classWt_1_1Chart_1_1WAxis_a59fd4c6a79f6e3ea75f21aa44c4acfa8}{set\+Visible()}} to change the visibility of an axis, \mbox{\hyperlink{classWt_1_1Chart_1_1WAxis_a79d70190d984c7f956e44b79de0af527}{set\+Grid\+Lines\+Enabled()}} to show grid lines for an axis. The pen styles for rendering the axis or grid lines may be changed using \mbox{\hyperlink{classWt_1_1Chart_1_1WAxis_ab1b9dcb9e23e01ec18994712fa5e11d1}{set\+Pen()}} and \mbox{\hyperlink{classWt_1_1Chart_1_1WAxis_aa97db5c1f1bff788382ef48b8a6d1cc2}{set\+Grid\+Lines\+Pen()}}. A margin between the axis and the main plot area may be configured using \mbox{\hyperlink{classWt_1_1Chart_1_1WAxis_a61a297a8d3c99e74b07e9def9ebe88a5}{set\+Margin()}}.

By default, the axis will automatically adjust its range so that all data will be visible. You may manually specify a range using \mbox{\hyperlink{classWt_1_1Chart_1_1WAxis_a878e20a87f7f6c5b4693bb81699c9f5b}{set\+Minimum()}}, set\+Maximum or \mbox{\hyperlink{classWt_1_1Chart_1_1WAxis_aae563fd94b81a03cfe2662dbde2ae548}{set\+Range()}}. The interval between labels is by default automatically adjusted depending on the axis length and the range, but may be manually specified using \mbox{\hyperlink{classWt_1_1Chart_1_1WAxis_a68b1f798fd0d9a713fac29f9d761e2a8}{set\+Label\+Interval()}}.

The axis has support for being \char`\"{}broken\char`\"{}, to support displaying data with a few outliers which would otherwise swamp the chart. This is not done automatically, but instead you need to use \mbox{\hyperlink{classWt_1_1Chart_1_1WAxis_a159f74342b4800e5c5be11eb1b2a351b}{set\+Break()}} to specify the value range that needs to be omitted from the axis. The omission is rendered in the axis and in bars that cross the break.

The labels are shown using a \char`\"{}\%.\+4g\char`\"{} format string for numbers, and a suitable format for \mbox{\hyperlink{group__charts_ggadfcaed8cd6ff52f5747f7df48f2eeab6a44749712dbec183e983dcd78a7736c41}{Axis\+Scale\+::\+Date}} or \mbox{\hyperlink{group__charts_ggadfcaed8cd6ff52f5747f7df48f2eeab6a8cf10d2341ed01492506085688270c1e}{Axis\+Scale\+::\+Date\+Time}} scales, based on heuristics. The format may be customized using \mbox{\hyperlink{classWt_1_1Chart_1_1WAxis_a7c37c241825af404b36916dcae9ae87b}{set\+Label\+Format()}}. The angle of the label text may be changed using \mbox{\hyperlink{classWt_1_1Chart_1_1WAxis_afd850eefc2beb7dffa5a637987cb1c6c}{set\+Label\+Angle()}}. By default, all labels are printed horizontally.
