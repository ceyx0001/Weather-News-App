class Wt\+::\+Dbo\+::query\+\_\+result\+\_\+traits$<$ Result $>$}

Traits class for result types. 

This traits class may be used to add support for using classes or structs as a result for a Session\+::query().

The library provides by default support for primitive types, using \mbox{\hyperlink{structWt_1_1Dbo_1_1sql__value__traits}{sql\+\_\+value\+\_\+traits}}, mapped objects held by ptr types, and std\+::tuple$<$$>$ or any combination of these.

Sets a result value. 

Sets the value at {\ttfamily index}, where index indicates the field whose value needs to be updated.

When {\ttfamily index} is out-\/of-\/bounds, it should be decremented with as many fields as there are in this result type. Otherwise, index should be set to -\/1 after the value has been set.
