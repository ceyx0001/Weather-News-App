This is an example that illustrates how {\ttfamily \mbox{\hyperlink{namespaceWt_1_1Auth}{Wt\+::\+Auth}}} can be extended with additional authentication mechanisms. The example is introduced in \href{http://www.webtoolkit.eu/wt/blog/2012/02/17/qr_code_login_example}{\texttt{ this post}}

The example adds the ability to login by authenticating on a mobile phone after scanning a QR code. This means that if on the phone you have a remember-\/me cookie stored, you can sign in to a public computer without needing password authentication.\hypertarget{md_wt-4_88_82_examples_qrlogin_README_autotoc_md115}{}\doxysection{How to run}\label{md_wt-4_88_82_examples_qrlogin_README_autotoc_md115}
