\doxysubsection{Detailed Description}
A widget that provides in-\/place-\/editable text. 

The W\+In\+Place\+Edit provides a text that may be edited in place by the user by clicking on it. When clicked, the text turns into a line edit, with optionally a save and cancel button (see \mbox{\hyperlink{classWt_1_1WInPlaceEdit_aac994b226bc2c33ecb36032b8f488933}{set\+Buttons\+Enabled()}}).

When the user saves the edit, the \mbox{\hyperlink{classWt_1_1WInPlaceEdit_a9c060014e820680fa242e051ebb77595}{value\+Changed()}} signal is emitted.

All these widgets may be styled as such. It does not provide style information.
