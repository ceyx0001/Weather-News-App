\doxysubsection{Detailed Description}
A date picker. 

A date picker shows a line edit and an icon which when clicked popups a \mbox{\hyperlink{classWt_1_1WCalendar}{W\+Calendar}} for editing the date. Any date entered in the line edit is reflected in the calendar, and vice-\/versa.

Each of these widgets may be accessed individually (\mbox{\hyperlink{classWt_1_1WDatePicker_a08a1dd7cf98a5ac103093acf5d6bff33}{line\+Edit()}}, \mbox{\hyperlink{classWt_1_1WDatePicker_a1b031904fc611b91d00bb47a04c1b1b2}{calendar()}}, and \mbox{\hyperlink{classWt_1_1WDatePicker_aa6a65df4aba99d42c0696d8e858805d4}{display\+Widget()}}) and there is a constructor that allows you to specify an existing line edit and display widget.

The date format used by default is {\ttfamily \char`\"{}dd/\+M\+M/yyyy\char`\"{}} and can be changed using \mbox{\hyperlink{classWt_1_1WDatePicker_a2ba8a8b7bae453b92e4a78725ed2f0ce}{set\+Format()}}. At any time, the date set may be read using \mbox{\hyperlink{classWt_1_1WDatePicker_a4920f662e74bef3a2ac5400f6c876869}{date()}}, or can be changed using \mbox{\hyperlink{classWt_1_1WDatePicker_adc47d5a1b0faf43dfed193f22f93c559}{set\+Date()}}.

\doxyparagraph*{i18n}

Internationalization of \mbox{\hyperlink{classWt_1_1WDatePicker}{W\+Date\+Picker}} is mostly handled through the internationalization mechanism of \mbox{\hyperlink{classWt_1_1WDate}{W\+Date}}.
