A function that renders a macro block as long as the given condition is true. 

The function will consider the first argument as the condition, and the second argument as the key for a localized string that is a macro block.

Just like the \mbox{\hyperlink{structWt_1_1WTemplate_1_1Functions_a3c7e34998d4171b707d154f96343600c}{block()}} function, you can provide additional arguments, so the third argument will be what is filled in into {\ttfamily \{1\}} in the macro block, etc.
