\doxysubsection{Detailed Description}
An input control for integer numbers. 

The spin box provides a control for entering an integer number. It consists of a line edit, and buttons which allow to increase or decrease the value. If you rather need input of a fractional number, use \mbox{\hyperlink{classWt_1_1WDoubleSpinBox}{W\+Double\+Spin\+Box}} instead.

W\+Spin\+Box is an \mbox{\hyperlink{classWt_1_1WWidget_ac78e3af143883334c82031790c87416e}{inline }} widget.

\begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{classWt_1_1WDoubleSpinBox}{W\+Double\+Spin\+Box}}
\end{DoxySeeAlso}
\begin{DoxyNote}{Note}
A spinbox configures a validator for validating the input. Therefore you cannot set a validator yourself. 
\end{DoxyNote}
