Returns whether or not the database column is used in the model. 

We need to make sure the database column is actually used in the model before storing the information in the \mbox{\hyperlink{classWt_1_1WFormModel}{W\+Form\+Model}}. Otherwise \mbox{\hyperlink{classWt_1_1WFormModel_a6d52d3ce5772935b069265a1c44f2146}{W\+Form\+Model\+::set\+Value}} will add the column to the model, which is not what we want here.
