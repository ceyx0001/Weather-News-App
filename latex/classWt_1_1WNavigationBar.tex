\doxysubsection{Detailed Description}
A navigation bar. 

\begin{DoxyNote}{Note}
\mbox{\hyperlink{classWt_1_1WNavigationBar}{W\+Navigation\+Bar}} is currently only styled in the Bootstrap themes.

When using \mbox{\hyperlink{classWt_1_1WBootstrap5Theme}{W\+Bootstrap5\+Theme}}, no color schemes are applied by default. You will have to add a \char`\"{}bg-\/\char`\"{} style class, and \char`\"{}navbar-\/light\char`\"{} or \char`\"{}navbar-\/dark\char`\"{} yourself. See the \href{https://getbootstrap.com/docs/5.1/components/navbar/\#color-schemes}{\texttt{ Bootstrap documentation on navbar color schemes}} for more info. 
\end{DoxyNote}
