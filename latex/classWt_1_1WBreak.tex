\doxysubsection{Detailed Description}
A widget that provides a line break between inline widgets. 

This is an \mbox{\hyperlink{classWt_1_1WWidget_ac78e3af143883334c82031790c87416e}{inline }} widget that provides a line break inbetween its sibling widgets (such as \mbox{\hyperlink{classWt_1_1WText}{W\+Text}}).
