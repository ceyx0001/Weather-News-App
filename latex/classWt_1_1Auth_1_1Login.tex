\doxysubsection{Detailed Description}
A class that manages the current login state. 

This is a model class which is typically associated with a single session, for the duration of the session.

Widgets that implement authentication (and thus produce authentication changes), will indicate their result in this object using the \mbox{\hyperlink{classWt_1_1Auth_1_1Login_af3993f0f74c0deea96cd5cc62cdc6749}{login()}} or \mbox{\hyperlink{classWt_1_1Auth_1_1Login_af10e98871028d79460a4e64415ca90e4}{logout()}} methods.

Widgets that want to react to login state changes (typically, as user logging in or out) should listen to the \mbox{\hyperlink{classWt_1_1Auth_1_1Login_aafb1d791aeee8b59d28faeda88e8392b}{changed()}} signal of this object.
