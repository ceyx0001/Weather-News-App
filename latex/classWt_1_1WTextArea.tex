\doxysubsection{Detailed Description}
A widget that provides a multi-\/line edit. 

To act upon text changes, connect a slot to the \mbox{\hyperlink{classWt_1_1WFormWidget_a880589c79f1b93e7f38330e28cf6f5e2}{changed()}} signal. This signal is emitted when the user changed the content, and subsequently removes the focus from the line edit.

To act upon editing, connect a slot to the \mbox{\hyperlink{classWt_1_1WInteractWidget_aed28d2568889f54a4809250ae265a08b}{key\+Went\+Up()}} signal.

At all times, the current content may be accessed with the \mbox{\hyperlink{classWt_1_1WTextArea_a4b347ba0b03ad5c06d21866e484ee302}{text()}} method.

W\+Text\+Area is an \mbox{\hyperlink{classWt_1_1WWidget_ac78e3af143883334c82031790c87416e}{inline }} widget.

\doxyparagraph*{C\+SS}

The widget corresponds to an H\+T\+ML {\ttfamily $<$textarea$>$} tag can be styled using inline or external C\+SS as appropriate. The empty\+Text style can be configured via .Wt-\/edit-\/empty\+Text.

This is used by the layout managers to correct for a built-\/in border which interferes with setting a widget\textquotesingle{}s width (or height) to 100\%.

A layout manager needs to set the width to 100\% only for form widgets (\mbox{\hyperlink{classWt_1_1WTextArea}{W\+Text\+Area}}, \mbox{\hyperlink{classWt_1_1WLineEdit}{W\+Line\+Edit}}, \mbox{\hyperlink{classWt_1_1WComboBox}{W\+Combo\+Box}}, etc...). For form widgets, the border width depends on the specific browser/platform combination, unless an explicit border is set for the widget.

When setting an explicit border for the widget using a style class, you will want to reimplement this method to return this border width, in case you want to set the widget inside a layout manager.

Unlike the \mbox{\hyperlink{classWt_1_1WFormWidget_a880589c79f1b93e7f38330e28cf6f5e2}{changed()}} signal, the signal is fired on every change, not only when the focus is lost. Unlike the \mbox{\hyperlink{classWt_1_1WInteractWidget_a9ddbec77fbb35319948fa8436f1aa2d4}{key\+Pressed()}} signal, this signal is fired also for other events that change the text, such as paste actions.
