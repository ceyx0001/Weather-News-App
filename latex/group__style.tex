\doxysubsection{Detailed Description}
Collection of classes for markup of widgets. 

The recommended way to provide style information for your Wt application is using C\+SS stylesheets. You may add rules to the inline style sheet using \mbox{\hyperlink{classWt_1_1WCssStyleSheet_a353021b958cc73060b1c8b5e6c2dadcb}{W\+Css\+Style\+Sheet\+::add\+Rule()}}, or by linking one or more external stylesheets using \mbox{\hyperlink{classWt_1_1WApplication_a8dcc025cd58968e087de8e90e6a4f547}{W\+Application\+::use\+Style\+Sheet()}}.

Alternatively, you may also provide style information directly, using \mbox{\hyperlink{classWt_1_1WCssDecorationStyle}{W\+Css\+Decoration\+Style}}, which you can manipulate for each widget using \mbox{\hyperlink{classWt_1_1WWidget_ac1833c7c01599b3733712ab0bf3c3a0a}{W\+Widget\+::decoration\+Style()}}.
