\doxysubsection{Detailed Description}
A resource response continuation object. 

A response continuation object is used to keep track of a response which is to be continued.

You may associate data with the object using \mbox{\hyperlink{classWt_1_1Http_1_1ResponseContinuation_a018939ff7a622576f072b94de59d2b6e}{set\+Data()}}.

A continuation is used to resume sending more data later for this response. There are two possible reasons for this\+:
\begin{DoxyItemize}
\item the entire response is quite big and you may want to read and send it in smaller chunks. This avoids unbounded memory usage since the I/O layer buffers the response first in memory to send it then out to a possibly slow client using async I/O.
\item you may not have any more data available, currently, but expect more data later. In that case you can call \mbox{\hyperlink{classWt_1_1Http_1_1ResponseContinuation_a3b441e3fa71a68b9dfb7091464e07433}{wait\+For\+More\+Data()}} and later call \mbox{\hyperlink{classWt_1_1WResource_a1a9c35416eb690939ff288b053da3e1b}{W\+Resource\+::have\+More\+Data()}} when more data is available.
\end{DoxyItemize}
