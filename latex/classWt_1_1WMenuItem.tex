\doxysubsection{Detailed Description}
A single item in a menu. 

Since Wt 3.\+3.\+0, this item is now a proper widget, which renders a single item in a menu.

An optional contents item can be associated with a menu item, which is inserted and shown in the widget stack of the menu to which this menu item belongs.

\doxyparagraph*{C\+SS}

A menu item renders as a $>$li\&;lt with additional markup/style classes provided by the theme. Unless you use the bootstrap theme, you will need to provide appropriate C\+SS.

Sets the path component for this item. 

The path component is used by the menu item in the application internal path (see \mbox{\hyperlink{classWt_1_1WApplication_a2c1a10aadc0d7ed877b5715b42ca4911}{W\+Application\+::set\+Internal\+Path()}}), when internal paths are enabled (see \mbox{\hyperlink{classWt_1_1WMenu_ae1807eba0b1002808dbec779d00a79dc}{W\+Menu\+::set\+Internal\+Path\+Enabled()}}) for the menu.

You may specify an empty {\ttfamily path} to let a menu item be the \char`\"{}default\char`\"{} menu option.

For example, if \mbox{\hyperlink{classWt_1_1WMenu_a534df522539f1f7c5a1a224acda48d56}{W\+Menu\+::internal\+Base\+Path()}} is {\ttfamily \char`\"{}/examples/\char`\"{}} and \mbox{\hyperlink{classWt_1_1WMenuItem_afded7ede14b03c42cfec06f9de515e6d}{path\+Component()}} for is {\ttfamily \char`\"{}charts/\char`\"{}}, then the internal path for the item will be {\ttfamily \char`\"{}/examples/charts/\char`\"{}}.

By default, the path is automatically derived from \mbox{\hyperlink{classWt_1_1WMenuItem_acc7baf68f04e05ff1af7caaf1da33202}{text()}}. If a \mbox{\hyperlink{classWt_1_1WString_a30835c468542b83d3d0a78a01e113669}{literal text}} is used, the path is based on the text itself, otherwise on the \mbox{\hyperlink{classWt_1_1WString_af9f2ed20d12c17f32c7c29683a326525}{key}}. It is converted to lower case, and replacing whitespace and special characters with \textquotesingle{}\+\_\+\textquotesingle{}.
