\doxysubsection{Detailed Description}
A J\+S\+ON object. 

This class represents a J\+S\+ON object, which defines a value map. It is implemented as a {\ttfamily std\+::map$<$std\+::string,\mbox{\hyperlink{classWt_1_1Json_1_1Value}{Json\+::\+Value}}$>$}. It is therefore possible to use the std\+::map A\+PI to deal with it.

However, a number of additional methods have been added to more conveniently query the existence and the type of contained objects. A \mbox{\hyperlink{classWt_1_1Json_1_1Object_aac177beb3cd68a18a892981712cab7f6}{get()}} method can be used to return a member value, which returns \mbox{\hyperlink{classWt_1_1Json_1_1Value_a344e20a378d9e91effbf91772b409942}{Value\+::\+Null}} if the member is not defined.
