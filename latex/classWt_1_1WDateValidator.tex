\doxysubsection{Detailed Description}
A validator for date input. 

This validator accepts input in the given date format, and optionally checks if the date is within a given range.

\doxyparagraph*{i18n}

The strings used in the \mbox{\hyperlink{classWt_1_1WDateValidator}{W\+Date\+Validator}} can be translated by overriding the default values for the following localization keys\+:
\begin{DoxyItemize}
\item Wt.\+W\+Date\+Validator.\+Date\+Too\+Early\+: The date must be after \{1\}
\item Wt.\+W\+Date\+Validator.\+Date\+Too\+Late\+: The date must be before \{1\}
\item Wt.\+W\+Date\+Validator.\+Wrong\+Date\+Range\+: The date must be between \{1\} and \{2\}
\item Wt.\+W\+Date\+Validator.\+Wrong\+Format\+: Must be a date in the format \textquotesingle{}\{1\}\textquotesingle{} 
\end{DoxyItemize}
