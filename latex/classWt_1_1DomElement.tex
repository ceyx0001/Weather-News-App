\doxysubsection{Detailed Description}
Class to represent a client-\/side D\+OM element (proxy). 

The D\+OM element proxy object is used as an intermediate layer to render the creation of new D\+OM elements or updates to existing D\+OM elements. A D\+OM element can be serialized to H\+T\+ML or to Java\+Script manipulations, and therefore is the main abstraction layer to avoid hard-\/coding Java\+Script-\/based rendering within the library while still allowing fine-\/grained Ajax updates or large-\/scale H\+T\+ML changes.

This is an internal A\+PI, subject to change.
