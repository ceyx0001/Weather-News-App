\doxysubsection{Detailed Description}
A widget which group widgets into a frame with a title. 

This is typically used in a form to group certain form elements together.

Usage example\+:

Like \mbox{\hyperlink{classWt_1_1WContainerWidget}{W\+Container\+Widget}}, W\+Group\+Box is by default displayed as a \mbox{\hyperlink{classWt_1_1WWidget_ac78e3af143883334c82031790c87416e}{block}}.



\doxyparagraph*{C\+SS}

The widget corresponds to the H\+T\+ML {\ttfamily $<$fieldset$>$} tag, and the title in a nested {\ttfamily $<$legend$>$} tag. This widget does not provide styling, and can be styled using inline or external C\+SS as appropriate.

The refresh method is invoked when the locale is changed using \mbox{\hyperlink{classWt_1_1WApplication_ac0af7e0131c926c77b207595c8c6e283}{W\+Application\+::set\+Locale()}} or when the user hit the refresh button.

The widget must actualize its contents in response.

This does {\bfseries{not}} rerender the widget! Calling \mbox{\hyperlink{classWt_1_1WGroupBox_ac3d412f4685ba75ce68334c17e884ca5}{refresh()}} usually does not have any effect (unless you\textquotesingle{}ve reimplemented \mbox{\hyperlink{classWt_1_1WGroupBox_ac3d412f4685ba75ce68334c17e884ca5}{refresh()}} to attach to it an effect).
