\doxysubsection{Detailed Description}
Abstract base class for an S\+QL connection. 

An sql connection manages a single connection to a database. It also manages a map of previously prepared statements indexed by id\textquotesingle{}s.

This class is part of \mbox{\hyperlink{namespaceWt_1_1Dbo}{Wt\+::\+Dbo}}\textquotesingle{}s backend A\+PI, and should not be used directly.

All methods will throw an exception if they could not be completed.
