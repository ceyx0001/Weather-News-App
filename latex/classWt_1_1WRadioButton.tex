\doxysubsection{Detailed Description}
A user control that represents a radio button. 

Use a \mbox{\hyperlink{classWt_1_1WButtonGroup}{W\+Button\+Group}} to group together radio buttons that reflect options that are mutually exclusive.

Usage example\+:

W\+Radio\+Button is an \mbox{\hyperlink{classWt_1_1WWidget_ac78e3af143883334c82031790c87416e}{inline }} widget.

\doxyparagraph*{C\+SS}

This widget corresponds to the H\+T\+ML {\ttfamily $<$input type=\char`\"{}radio\char`\"{}$>$} tag. When a label is specified, the input element is nested in a {\ttfamily $<$label$>$}.

This widget does not provide styling, and can be styled using inline or external C\+SS as appropriate.
