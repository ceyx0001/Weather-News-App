\doxysubsection{Detailed Description}
A value class to play a sound effect. 

This class provides a way to play an M\+P3 sound asynchonously (if the browser supports this). It is intended as a simple way to play event sounds (not quite for a media center).

This class uses a \mbox{\hyperlink{classWt_1_1WMediaPlayer}{W\+Media\+Player}} to play the sound (using H\+T\+ML $<$audio$>$ or a flash player).

{\footnotesize\ttfamily void Wt\+::\+W\+Sound\+::play (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Start asynchronous playback of the sound. 

This method returns immediately. It will cause the sound to be played for the configured amount of \mbox{\hyperlink{classWt_1_1WSound_aac5067b17c764fb792a3558a44c4e6fa}{loops()}}.

The behavior of \mbox{\hyperlink{classWt_1_1WSound_a6ca6b48ebc9e49ee6745228961df3817}{play()}} when a sound is already playing is undefind\+: it may be intermixed, sequentially queued, or a current playing sound may be stopped. It is recommended to call \mbox{\hyperlink{classWt_1_1WSound_a7c3ce5dc7204f5c156bffd71487e3f72}{stop()}} before \mbox{\hyperlink{classWt_1_1WSound_a6ca6b48ebc9e49ee6745228961df3817}{play()}} if you want to avoid mixing multiple instances of a single \mbox{\hyperlink{classWt_1_1WSound}{W\+Sound}} object. \mbox{\Hypertarget{classWt_1_1WSound_a6cf72578210c7a6ff345372afa85d198}\label{classWt_1_1WSound_a6cf72578210c7a6ff345372afa85d198}}
