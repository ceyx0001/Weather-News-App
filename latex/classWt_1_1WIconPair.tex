\doxysubsection{Detailed Description}
A widget that shows one of two icons depending on its state. 

This is a utility class that simply manages two images, only one of which is shown at a single time, which reflects the current \textquotesingle{}state\textquotesingle{}.

The widget may react to click events, by changing state.

\doxyparagraph*{C\+SS}

This widget does not provide styling, and can be styled using inline or external C\+SS as appropriate. The image may be styled via the {\ttfamily $<$img$>$} elements.
