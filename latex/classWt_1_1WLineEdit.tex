\doxysubsection{Detailed Description}
A widget that provides a single line edit. 

To act upon text changes, connect a slot to the \mbox{\hyperlink{classWt_1_1WFormWidget_a880589c79f1b93e7f38330e28cf6f5e2}{changed()}} signal. This signal is emitted when the user changed the content, and subsequently removes the focus from the line edit.

To act upon editing, connect a slot to the \mbox{\hyperlink{classWt_1_1WInteractWidget_aed28d2568889f54a4809250ae265a08b}{key\+Went\+Up()}} signal because the \mbox{\hyperlink{classWt_1_1WInteractWidget_a9ddbec77fbb35319948fa8436f1aa2d4}{key\+Pressed()}} signal is fired before the line edit has interpreted the keypress to change its text.

At all times, the current content may be accessed with the \mbox{\hyperlink{classWt_1_1WLineEdit_a18c393149e30024b22427a944d877094}{text()}} method.

You may specify a maximum length for the input using \mbox{\hyperlink{classWt_1_1WLineEdit_aaeca334d8a47be029b6a44912b01ed5d}{set\+Max\+Length()}}. If you wish to provide more detailed input validation, you may set a validator using the \mbox{\hyperlink{classWt_1_1WFormWidget_a8b5fa8567e930691b95f1d0fa69359f1}{set\+Validator(const std\+::shared\+\_\+ptr$<$\+W\+Validator$>$ \&)}} method. Validators provide, in general, both client-\/side validation (as visual feed-\/back only) and server-\/side validation when calling \mbox{\hyperlink{classWt_1_1WLineEdit_ac07b136a2c39130731ef79de9b0a1f5c}{validate()}}.

The widget corresponds to the H\+T\+ML {\ttfamily $<$input type=\char`\"{}text\char`\"{}$>$} or {\ttfamily $<$input type=\char`\"{}password\char`\"{}$>$} tag.

W\+Line\+Edit is an \mbox{\hyperlink{classWt_1_1WWidget_ac78e3af143883334c82031790c87416e}{inline }} widget.

\doxyparagraph*{C\+SS}

Event signal emitted when the text in the input field changed. 

This signal is emitted whenever the text contents has changed. Unlike the \mbox{\hyperlink{classWt_1_1WFormWidget_a880589c79f1b93e7f38330e28cf6f5e2}{changed()}} signal, the signal is fired on every change, not only when the focus is lost. Unlike the \mbox{\hyperlink{classWt_1_1WInteractWidget_a9ddbec77fbb35319948fa8436f1aa2d4}{key\+Pressed()}} signal, this signal is fired also for other events that change the text, such as paste actions.
