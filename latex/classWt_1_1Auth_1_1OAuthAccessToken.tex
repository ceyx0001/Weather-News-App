\doxysubsection{Detailed Description}
An O\+Auth access token. 

A access token is the result of an authorization process, and encapsulates the authorization to access protected information.

Next to its \mbox{\hyperlink{classWt_1_1Auth_1_1OAuthAccessToken_ad98c896c01fc3ebdcbf2487f17381590}{value()}}, it also contains optionally an expires date and a refresh token.
