\doxysubsection{Detailed Description}
Classes that handle the H\+T\+TP protocol. 

The Wt\+::Http namespace contains classes that deal with the H\+T\+TP protocol. This can be split in two groups\+:
\begin{DoxyItemize}
\item classes involved which deal with the server-\/side aspects of Wt through \mbox{\hyperlink{classWt_1_1WResource}{W\+Resource}}\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{classWt_1_1Http_1_1Request}{Request}}\+: a resource request
\item \mbox{\hyperlink{classWt_1_1Http_1_1Response}{Response}}\+: a resource response
\item \mbox{\hyperlink{classWt_1_1Http_1_1ResponseContinuation}{Response\+Continuation}}\+: an asynchronous response continuation object
\item \mbox{\hyperlink{classWt_1_1Http_1_1UploadedFile}{Uploaded\+File}}\+: a file uploaded in the \mbox{\hyperlink{classWt_1_1Http_1_1Request}{Request}}
\end{DoxyItemize}
\item classes that implement an H\+T\+TP client\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{classWt_1_1Http_1_1Client}{Client}}\+: an H\+T\+TP client
\item \mbox{\hyperlink{classWt_1_1Http_1_1Message}{Message}}\+: a message to be sent with the client, or received from the client. 
\end{DoxyItemize}
\end{DoxyItemize}
