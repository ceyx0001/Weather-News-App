\doxysubsection{Detailed Description}
A calendar date and clock time. 

The date time class combines the functionality of a \mbox{\hyperlink{classWt_1_1WDate}{W\+Date}} (for a calendar date) and \mbox{\hyperlink{classWt_1_1WTime}{W\+Time}} (for clock time) into a single class.

This class stores and represents the date time in U\+TC. To deal with local (wall clock) time, see \mbox{\hyperlink{classWt_1_1WLocalDateTime}{W\+Local\+Date\+Time}}. To convert from U\+TC to local time use \mbox{\hyperlink{classWt_1_1WDateTime_a94fc40589603a414424e7e836f36755b}{to\+Local\+Time()}}.

\doxyparagraph*{i18n}

The strings returned by \mbox{\hyperlink{classWt_1_1WDateTime_a2269c77624781225e7ea63322be767d0}{to\+String()}} and \mbox{\hyperlink{classWt_1_1WDateTime_a69ebd895f54465f7ee07b2b5c1047ba9}{time\+To()}} can be localized. If the thread using a \mbox{\hyperlink{classWt_1_1WDateTime}{W\+Date\+Time}} is bound to a \mbox{\hyperlink{classWt_1_1WApplication}{W\+Application}} (i.\+e. \mbox{\hyperlink{classWt_1_1WApplication_a27690e10fb1dffef9b688b5a6ecf26b0}{W\+Application\+::instance()}} does not return 0), the strings can be localized by overriding the default values for the localized string keys in the resource bundles of the \mbox{\hyperlink{classWt_1_1WApplication}{W\+Application}}. If the thread using a \mbox{\hyperlink{classWt_1_1WDateTime}{W\+Date\+Time}} is not bound to a \mbox{\hyperlink{classWt_1_1WApplication}{W\+Application}} (i.\+e. \mbox{\hyperlink{classWt_1_1WApplication_a27690e10fb1dffef9b688b5a6ecf26b0}{W\+Application\+::instance()}} returns 0), English strings will be used.

For \mbox{\hyperlink{classWt_1_1WDateTime_a2269c77624781225e7ea63322be767d0}{to\+String()}}, localization is handled through the i18n functionality of the \mbox{\hyperlink{classWt_1_1WDate}{W\+Date}} class. An invalid \mbox{\hyperlink{classWt_1_1WDateTime}{W\+Date\+Time}} is converted to a string as \char`\"{}\+Null\char`\"{}\+:
\begin{DoxyItemize}
\item Wt.\+W\+Date\+Time.\+null\+: Null
\end{DoxyItemize}

The \mbox{\hyperlink{classWt_1_1WDateTime_a69ebd895f54465f7ee07b2b5c1047ba9}{time\+To()}} method contains the following localized strings\+:
\begin{DoxyItemize}
\item Wt.\+W\+Date\+Time.\+seconds
\item Wt.\+W\+Date\+Time.\+minutes
\item and so on
\end{DoxyItemize}

The default translations of Wt.\+W\+Date\+Time.\+seconds are \char`\"{}one second\char`\"{} and \char`\"{}\{1\} seconds\char`\"{}. The \mbox{\hyperlink{classWt_1_1WString_af43284d8db7ade763739ac11555bae3b}{W\+String\+::trn()}} function is used for the translation, so different strings can be provided for different plural forms. The placeholder \{1\} will be replaced by the actual number of seconds. The same keys also exist for minutes, hours, days, weeks, months and years.
