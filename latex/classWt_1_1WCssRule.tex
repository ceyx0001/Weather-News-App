\doxysubsection{Detailed Description}
Abstract rule in a C\+SS style sheet. 

A rule presents C\+SS style properties that are applied to a selected set of elements.

Use \mbox{\hyperlink{classWt_1_1WCssTemplateRule}{W\+Css\+Template\+Rule}} if you would like to use a widget as a template for specifying ({\itshape and} updating) a style rule, using the widgets style properties, or \mbox{\hyperlink{classWt_1_1WCssTextRule}{W\+Css\+Text\+Rule}} if you wish to directly specify the C\+SS declarations.
