\doxysubsection{Detailed Description}
A Postgre\+S\+QL connection. 

This class provides the backend implementation for Postgre\+S\+QL databases.

When applicable, exceptions from the backend will return the five-\/character S\+Q\+L\+S\+T\+A\+TE error codes, as in \href{http://www.postgresql.org/docs/8.1/static/errcodes-appendix.html,}{\texttt{ http\+://www.\+postgresql.\+org/docs/8.\+1/static/errcodes-\/appendix.\+html,}} in \mbox{\hyperlink{classWt_1_1Dbo_1_1Exception_aec872dea5420267dc96b1b03c8193c49}{Exception\+::code()}}.
