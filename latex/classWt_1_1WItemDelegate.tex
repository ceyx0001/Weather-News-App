\doxysubsection{Detailed Description}
Standard delegate class for rendering a view item. 

This class provides the standard implementation for rendering an item (as in a \mbox{\hyperlink{classWt_1_1WAbstractItemView}{W\+Abstract\+Item\+View}}), and renders data provided by the standard data roles (see \mbox{\hyperlink{classWt_1_1ItemDataRole}{Item\+Data\+Role}}). It also provides default editing support using a line edit.

You may provide special editing support for an item by specializing the widget and reimplement \mbox{\hyperlink{classWt_1_1WItemDelegate_ac099ea4a441059523831939353ed6cb7}{create\+Editor()}}, \mbox{\hyperlink{classWt_1_1WItemDelegate_a3746136f103c4ad4d1574ac3f2ad11d4}{set\+Model\+Data()}}, \mbox{\hyperlink{classWt_1_1WItemDelegate_a79941ae50687f0e65da2f58f86bf571c}{edit\+State()}}, and \mbox{\hyperlink{classWt_1_1WItemDelegate_a9b15fa5d2feb767deb4233d73cb8d1e1}{set\+Edit\+State()}}.

Creates or updates a widget that renders an item. 

The following properties of an item are rendered\+:


\begin{DoxyItemize}
\item text using the \mbox{\hyperlink{classWt_1_1ItemDataRole_ae3ecc48a21764c9fce36a9be37ccc2c1}{Wt\+::\+Item\+Data\+Role\+::\+Display}} data, with the format specified by \mbox{\hyperlink{classWt_1_1WItemDelegate_a5c316445f87af6d053bb240ad6b295d6}{set\+Text\+Format()}}
\item a check box depending on the Wt\+::\+Item\+Flag\+::\+User\+Checkable flag and \mbox{\hyperlink{classWt_1_1ItemDataRole_af9f0b694aa760202db896dda696bf55f}{Wt\+::\+Item\+Data\+Role\+::\+Checked}} data
\item an anchor depending on the value of \mbox{\hyperlink{classWt_1_1ItemDataRole_ae478217452f058add8fd204ddacde8ae}{Wt\+::\+Item\+Data\+Role\+::\+Link}}
\item an icon depending on the value of \mbox{\hyperlink{classWt_1_1ItemDataRole_a9db6daf35b3c68db6df624411420d4b7}{Wt\+::\+Item\+Data\+Role\+::\+Decoration}}
\item a tooltip depending on the value of \mbox{\hyperlink{classWt_1_1ItemDataRole_a9013847e572f2b118b5b1a6fa74691a6}{Wt\+::\+Item\+Data\+Role\+::\+Tool\+Tip}}
\item a custom style class depending on the value of \mbox{\hyperlink{classWt_1_1ItemDataRole_ac91a7ef0ee5b9c13fd74ce2a1c57ecb9}{Wt\+::\+Item\+Data\+Role\+::\+Style\+Class}}
\end{DoxyItemize}

When the flags indicates Wt\+::\+View\+Item\+Render\+Flag\+::\+Editing, then \mbox{\hyperlink{classWt_1_1WItemDelegate_ac099ea4a441059523831939353ed6cb7}{create\+Editor()}} is called to create a suitable editor for editing the item.
