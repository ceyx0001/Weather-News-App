\doxysubsection{Detailed Description}
A widget which allows a user to choose a new password. 

This widget lets a user choose a new password.

The widget renders the {\ttfamily \char`\"{}\+Wt.\+Auth.\+template.\+update-\/password\char`\"{}} template. Optionally, it asks for the current password, as well as a new password.
