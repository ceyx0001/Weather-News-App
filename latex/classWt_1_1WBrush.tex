\doxysubsection{Detailed Description}
A value class that defines the style for filling a path. 

A brush defines the properties of how areas (the interior of shapes) are filled. A brush is defined either as a solid color or a gradient.

\doxyparagraph*{Java\+Script exposability}

A W\+Brush is Java\+Script exposable. If a W\+Brush \mbox{\hyperlink{classWt_1_1WJavaScriptExposableObject_a4e121ccbb14715c33dc1b019b6636156}{is Java\+Script bound}}, it can be accessed in your custom Java\+Script code through \mbox{\hyperlink{classWt_1_1WJavaScriptHandle_a0d0bc00f0bcafb596c3f6c96c4241ded}{its handle\textquotesingle{}s js\+Ref()}}. At the moment, only the \mbox{\hyperlink{classWt_1_1WBrush_a90b7f01b19a3bf9b4f50f18981fc9d67}{color()}} property is exposed, e.\+g. a brush with the color W\+Color(10,20,30,255) will be represented in Java\+Script as\+:

\DoxyCodeLine{\{}
\DoxyCodeLine{  \mbox{\hyperlink{classWt_1_1WBrush_a90b7f01b19a3bf9b4f50f18981fc9d67}{color}}: [10,20,30,255]}
\DoxyCodeLine{\}}
\end{DoxyCode}


\begin{DoxyWarning}{Warning}
A W\+Brush that is Java\+Script exposed should be modified only through its \mbox{\hyperlink{classWt_1_1WJavaScriptHandle}{handle}}. Any attempt at modifying it will cause an exception to be thrown.
\end{DoxyWarning}
