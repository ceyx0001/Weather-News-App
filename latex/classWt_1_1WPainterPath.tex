\doxysubsection{Detailed Description}
A path defining a shape. 

A painter path represents a (complex) path that may be composed of lines, arcs and bezier curve segments, and painted onto a paint device using \mbox{\hyperlink{classWt_1_1WPainter_a2ef57868d6fbf99a901cbe1f29723b50}{W\+Painter\+::draw\+Path()}}.

The path that is composed in a painter path may consist of multiple closed sub-\/paths. Only the last sub-\/path can be left open.

To compose a path, this class maintains a current position, which is the starting point for the next drawing operation. An operation may draw a line (see \mbox{\hyperlink{classWt_1_1WPainterPath_ac52bacd9bb3b07043e56be2af56f7f66}{line\+To()}}), arc (see \mbox{\hyperlink{classWt_1_1WPainterPath_aa32b8c9d89712a5cad37325cbba099c0}{arc\+To()}}), or bezier curve (see \mbox{\hyperlink{classWt_1_1WPainterPath_a5cf7125271a74e15a52db05876b8c878}{quad\+To()}} and \mbox{\hyperlink{classWt_1_1WPainterPath_ac5a13ec3faf642cd05a57988366e005f}{cubic\+To()}}) from the current position to a new position. A new sub path may be started by moving the current position to a new location (see \mbox{\hyperlink{classWt_1_1WPainterPath_a9801a3988b2a57c412ba175142d2e3ff}{move\+To()}}), which automatically closes the previous sub path.

When sub paths overlap, the result is undefined (it is dependent on the underlying painting device).

\doxyparagraph*{Java\+Script exposability}

A W\+Painter\+Path is Java\+Script exposable. If a W\+Painter\+Path \mbox{\hyperlink{classWt_1_1WJavaScriptExposableObject_a4e121ccbb14715c33dc1b019b6636156}{is Java\+Script bound}}, it can be accessed in your custom Java\+Script code through \mbox{\hyperlink{classWt_1_1WJavaScriptHandle_a0d0bc00f0bcafb596c3f6c96c4241ded}{its handle\textquotesingle{}s js\+Ref()}}.

A W\+Painter\+Path is represented in Java\+Script as an array of segments, where each segment is defined by a three element array\+: \mbox{[}x,y,type\mbox{]}, where type is the integer representation of the type of a segment.

For example, a 10 by 10 square with the top left at (10,10) is represented as\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{[}
\DoxyCodeLine{ [10,10,0], \textcolor{comment}{// move to (10,10)}}
\DoxyCodeLine{ [20,10,1], \textcolor{comment}{// line to (20,10)}}
\DoxyCodeLine{ [20,20,1], \textcolor{comment}{// line to (20,20)}}
\DoxyCodeLine{ [10,20,1], \textcolor{comment}{// line to (10,20)}}
\DoxyCodeLine{ [10,10,1]  \textcolor{comment}{// line to (10,10)}}
\DoxyCodeLine{]}
\end{DoxyCode}


\begin{DoxyWarning}{Warning}
A W\+Painter\+Path that is Java\+Script exposed should be modified only through its \mbox{\hyperlink{classWt_1_1WJavaScriptHandle}{handle}}. Any attempt at modifying it will cause an exception to be thrown.
\end{DoxyWarning}
