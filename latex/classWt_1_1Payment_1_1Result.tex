\doxysubsection{Detailed Description}
A class that represents the result of a payment A\+PI call. 

This class indicates the result of an asynchronous call\+: \mbox{\hyperlink{classWt_1_1Payment_1_1Result_aceadb6ddd35c866c68ba4cf81e769e25}{error()}} indicates whether there was an error.
