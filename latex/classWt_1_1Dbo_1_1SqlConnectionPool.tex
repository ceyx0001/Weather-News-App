\doxysubsection{Detailed Description}
Abstract base class for a S\+QL connection pool. 

An sql connection pool manages a pool of connections. It is shared between multiple sessions to allow these sessions to use a connection while handling a transaction. \mbox{\hyperlink{classNote}{Note}} that a session only needs a connection while in-\/transaction, and thus you only need as much connections as the number of concurrent transactions.
