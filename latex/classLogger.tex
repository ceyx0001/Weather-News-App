\doxysubsection{Detailed Description}
Manages \mbox{\hyperlink{classLogMessage}{Log\+Message}} objects. 

The \mbox{\hyperlink{classLogger}{Logger}} object saves log messages to a database and also is able to load all stored log messages from the database. It is a singleton using Scott Meyer\textquotesingle{}s implementation that allows only one instance of the object globally. \begin{DoxyAuthor}{Author}
Jun Shao 
\end{DoxyAuthor}
