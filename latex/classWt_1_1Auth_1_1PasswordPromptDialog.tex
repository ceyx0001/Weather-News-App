\doxysubsection{Detailed Description}
A dialog that prompts for the user password. 

This is a simple dialog, useful for prompting the user to enter his password. This may be convenient for example to let the user upgrade from a weak authentication to a strong authentication.

The dialog uses a \mbox{\hyperlink{classWt_1_1Auth_1_1Login}{Login}} object to get the currently identified user, and also sets the result of the login process by calling \mbox{\hyperlink{classWt_1_1Auth_1_1Login_af3993f0f74c0deea96cd5cc62cdc6749}{Login\+::login()}} on this object.

The dialog renders the {\ttfamily \char`\"{}\+Wt.\+Auth.\+template.\+password-\/prompt\char`\"{}} template.
