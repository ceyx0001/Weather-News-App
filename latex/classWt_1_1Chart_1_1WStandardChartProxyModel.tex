\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classWt_1_1Chart_1_1WAbstractChartModel}{W\+Abstract\+Chart\+Model}} implementation that wraps a \mbox{\hyperlink{classWt_1_1WAbstractItemModel}{W\+Abstract\+Item\+Model}}. 

This model delegates all functions to an underlying \mbox{\hyperlink{classWt_1_1WAbstractItemModel}{W\+Abstract\+Item\+Model}}, using the appropriate roles.

This model also triggers the \mbox{\hyperlink{classWt_1_1Chart_1_1WAbstractChartModel_a8514b7e6fd279e57e3812cf1a364a3aa}{changed()}} signal whenever the underlying \mbox{\hyperlink{classWt_1_1WAbstractItemModel}{W\+Abstract\+Item\+Model}} is changed.
