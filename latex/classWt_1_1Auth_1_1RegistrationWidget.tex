\doxysubsection{Detailed Description}
A registration widget. 

This implements a widget which allows a new user to register. The widget renders the {\ttfamily \char`\"{}\+Wt.\+Auth.\+template.\+registration\char`\"{}} template. and uses a \mbox{\hyperlink{classWt_1_1Auth_1_1RegistrationModel}{Registration\+Model}} for the actual registration logic.

Typically, you may want to specialize this widget to ask for other information.
