\doxysubsection{Detailed Description}
A validator that validates integer user input. 

This validator checks whether user input is an integer number in a pre-\/defined range.

\doxyparagraph*{i18n}

The strings used in this class can be translated by overriding the default values for the following localization keys\+:
\begin{DoxyItemize}
\item Wt.\+W\+Int\+Validator.\+Not\+An\+Integer\+: Must be an integer number
\item Wt.\+W\+Int\+Validator.\+Too\+Small\+: The number must be larger than \{1\}
\item Wt.\+W\+Int\+Validator.\+Bad\+Range\+: The number must be in the range \{1\} to \{2\}
\item Wt.\+W\+Int\+Validator.\+Too\+Large\+: The number must be smaller than \{1\} 
\end{DoxyItemize}
