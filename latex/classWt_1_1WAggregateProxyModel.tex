\doxysubsection{Detailed Description}
A proxy model for Wt\textquotesingle{}s item models that provides column aggregation. 

This proxy model does not store data itself, but presents data from a source model, and presents methods to organize columns using aggregation, through which a user may navigate (usually to obtain more detailed results related to a single aggregate quantity).

To use this proxy model, you should provide a source model using \mbox{\hyperlink{classWt_1_1WAggregateProxyModel_a115b5c3431c1b86ac175efe49e13f89e}{set\+Source\+Model()}}, and define column ranges that can be aggregated using \mbox{\hyperlink{classWt_1_1WAggregateProxyModel_add590f7e0789812c73874ab2bd4161bf}{add\+Aggregate()}}.

\begin{DoxyNote}{Note}
This model does not support dynamic changes to the column definition of the source model (i.\+e. insertions or deletions of source model columns). 
\end{DoxyNote}

Adds a new column aggregation definition. 

The {\ttfamily parent\+Column} is the column index in the source model that acts as an aggregate for columns {\ttfamily first\+Column} to {\ttfamily last\+Column}. {\ttfamily parent\+Column} must border the range defined by {\ttfamily first\+Column} to {\ttfamily last\+Column\+:} 
\begin{DoxyCode}{0}
\DoxyCodeLine{parentColumn == firstColumn -\/ 1 || parentColumn == lastColumn + 1}
\end{DoxyCode}


\mbox{\hyperlink{classNote}{Note}} that column parameters reference column indexes in the source model.

Aggregation definitions can be nested, but should be strictly hierarchical.

The aggregate column will initially be collapsed.

Only one aggregate can be defined per {\ttfamily parent\+Column}. \mbox{\Hypertarget{classWt_1_1WAggregateProxyModel_a5999b2e18c05eceb6062c821a6643948}\label{classWt_1_1WAggregateProxyModel_a5999b2e18c05eceb6062c821a6643948}}
