\doxysubsection{Detailed Description}
Theme based on the Bootstrap 5 C\+SS framework. 

This theme implements support for building a Wt web application that uses Bootstrap 5 as a theme for its (layout and) styling.

Using this theme, various widgets provided by the library are rendered using markup that is compatible with Bootstrap 5. The bootstrap theme is also extended with a proper (compatible) styling of widgets for which bootstrap does not provide styling (table views, tree views, sliders, etc...).

By default, the theme will use C\+SS and Java\+Script resources that are shipped together with the Wt distribution, but you can replace the C\+SS with custom-\/built C\+SS by reimplementing \mbox{\hyperlink{classWt_1_1WBootstrap5Theme_acf061ac4968014a7188d248edbda7ef1}{style\+Sheets()}}.

Although this theme styles individual widgets correctly, for your web application\textquotesingle{}s layout you are recommended to use \mbox{\hyperlink{classWt_1_1WTemplate}{W\+Template}} in conjunction with Bootstrap\textquotesingle{}s C\+SS classes. For this we refer to Bootstrap\textquotesingle{}s documentation at \href{https://getbootstrap.com}{\texttt{ https\+://getbootstrap.\+com}}.
