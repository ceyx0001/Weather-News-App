\doxysubsection{Detailed Description}
A value class that represents a date on the Gregorian calendar. 

Class which holds a date on the gregorian calendar, specified as day/month/year.

A W\+Date may either specify a valid date, or be a {\itshape Null} date (using the default constructor \mbox{\hyperlink{classWt_1_1WDate_a88823f7de1cc16e9211015733f15eac3}{W\+Date\+::\+W\+Date()}}, for which \mbox{\hyperlink{classWt_1_1WDate_a15f10a5100d5865f0536f5e73e712851}{is\+Null()}} returns {\ttfamily true}).

A valid date may be specified by year, month, and day of month (using the \mbox{\hyperlink{classWt_1_1WDate_ae88e64cff5fa3f28283f26fa7c8e4b99}{W\+Date(year, month, day) }} constructor, or the \mbox{\hyperlink{classWt_1_1WDate_ad9f497db41a525c9955aa6c45fa9a692}{set\+Date()}} method). When attempting to specify an invalid date (with an impossible combination of year/month/date), is\+Ivalid() will return {\ttfamily false}.

The class provides a flexible way for converting between strings and dates. Use \mbox{\hyperlink{classWt_1_1WDate_a8fb0648e3910ee4fc68a55e245f86463}{to\+String()}} to convert to strings, and \mbox{\hyperlink{classWt_1_1WDate_a240437f713a7dbe3821798b5a8158b3b}{from\+String()}} for parsing strings. Both methods take a format string, and the same format syntax is supported by both methods.

Simple operations are supported to compare dates, or to calculate with dates.

\doxyparagraph*{i18n}

If the thread using a \mbox{\hyperlink{classWt_1_1WDate}{W\+Date}} is bound to a \mbox{\hyperlink{classWt_1_1WApplication}{W\+Application}} (i.\+e. \mbox{\hyperlink{classWt_1_1WApplication_a27690e10fb1dffef9b688b5a6ecf26b0}{W\+Application\+::instance()}} does not return 0), the date strings can be localized by overriding the default values for the localized string keys in the resource bundles of the \mbox{\hyperlink{classWt_1_1WApplication}{W\+Application}}\+:
\begin{DoxyItemize}
\item Short day names\+: Wt.\+W\+Date.\+Mon through Wt.\+W\+Date.\+Sun
\item Long day names\+: Wt.\+W\+Date.\+Monday through Wt.\+W\+Date.\+Sunday
\item Short month names\+: Wt.\+W\+Date.\+Jan through Wt.\+W\+Date.\+Dec
\item Long month names\+: Wt.\+W\+Date.\+January through Wt.\+W\+Date.\+December
\end{DoxyItemize}

If the thread using a \mbox{\hyperlink{classWt_1_1WDate}{W\+Date}} is not bound to a \mbox{\hyperlink{classWt_1_1WApplication}{W\+Application}} (i.\+e. \mbox{\hyperlink{classWt_1_1WApplication_a27690e10fb1dffef9b688b5a6ecf26b0}{W\+Application\+::instance()}} returns 0), english strings will be used.

Internationalization affects both \mbox{\hyperlink{classWt_1_1WDate}{W\+Date}} to string conversions and string to \mbox{\hyperlink{classWt_1_1WDate}{W\+Date}} conversion.
