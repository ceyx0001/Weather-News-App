\doxysubsection{Detailed Description}
A value class that defines a 2D point. 

\doxyparagraph*{Java\+Script exposability}

A W\+PointF is Java\+Script exposable. If a W\+PointF \mbox{\hyperlink{classWt_1_1WJavaScriptExposableObject_a4e121ccbb14715c33dc1b019b6636156}{is Java\+Script bound}}, it can be accessed in your custom Java\+Script code through \mbox{\hyperlink{classWt_1_1WJavaScriptHandle_a0d0bc00f0bcafb596c3f6c96c4241ded}{its handle\textquotesingle{}s js\+Ref()}}. A point is represented in Java\+Script as an array of two elements, e.\+g. a point W\+Point\+F(10,20) will be represented in Java\+Script as\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{[10, 20]}
\end{DoxyCode}


\begin{DoxyWarning}{Warning}
A W\+PointF that is Java\+Script exposed should be modified only through its \mbox{\hyperlink{classWt_1_1WJavaScriptHandle}{handle}}. Any attempt at modifying it will cause an exception to be thrown.
\end{DoxyWarning}
