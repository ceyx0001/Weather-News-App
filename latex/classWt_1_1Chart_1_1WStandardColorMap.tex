\doxysubsection{Detailed Description}
Standard color\+Map. 

The W\+Standard\+Color\+Map is defined by a list of value-\/to-\/color pairs. The W\+Standard\+Color\+Map has two modes\+: a continuous mode, in which the colors are linearly interpolated in between the pair values, and a non-\/continuous mode, where the values are not interpolated, so that the colormap has a banded effect. In non-\/continuous mode, the color of a given point P is the color of the pair with the largest value smaller than P.

Numerical values above the maximum value in the list map to the maximum value\textquotesingle{}s color, all values below the minimum value in the list map to the minimum value\textquotesingle{}s color. The range indicated by the minimum and maximum passed to the constructor determines which part of the colormap is drawn by \mbox{\hyperlink{classWt_1_1Chart_1_1WStandardColorMap_a853786132b346d719ce6e6f029dbb6b9}{create\+Strip()}} or \mbox{\hyperlink{classWt_1_1Chart_1_1WStandardColorMap_a1823407ef62c908d3686a71d2221979d}{paint\+Legend()}}.

The figure below illustrates the possible colormaps that can be constructed from the list \{\char`\"{}0.\+0 -\/ Standard\+Color\+::\+Dark\+Red\char`\"{}, \char`\"{}1.\+0 -\/ Standard\+Color\+::\+Red\char`\"{}, \char`\"{}2.\+0 -\/ Standard\+Color\+::\+Gray\char`\"{}\}. The discrete map (on the left) has the range \mbox{[}0, 3\mbox{]}, the continuous map (on the right) has the range \mbox{[}0, 2\mbox{]}. The utility method \mbox{\hyperlink{classWt_1_1Chart_1_1WStandardColorMap_a39cfdfa352116ee10e544ca8f684596a}{discretise()}} is also applied to the continuous colormap to obtain a colormap with 5 bands in the same range.
