\doxysubsection{Detailed Description}
A mail message. 

This class represents a M\+I\+M\+E-\/compliant mail message.

The message can have a plain text body and an optional H\+T\+ML body, which when present is encoded as an M\+I\+ME multipart/alternative. It is recommended to send the same contents both in a plain text and an H\+T\+ML variant.

\mbox{\hyperlink{structWt_1_1Mail_1_1Message_1_1Recipient}{Recipient}} names, names, and body text may contain unicode text.
