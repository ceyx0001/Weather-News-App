\doxysubsection{Detailed Description}
A selection box allows selection from a list of options. 

By default, a selection box may be used to let the user select one item from a list. This may be changed to multiple selection mode using \mbox{\hyperlink{classWt_1_1WSelectionBox_aa3119417024e25b7b24e66cab17edb31}{set\+Selection\+Mode()}}.

The current selection may be set and read using \mbox{\hyperlink{classWt_1_1WComboBox_a903e2d98c6860fe09198bdb43f0566bd}{set\+Current\+Index()}} and \mbox{\hyperlink{classWt_1_1WComboBox_a46d536faa379cf01e50974065265780f}{current\+Index()}}, for \mbox{\hyperlink{}{Selection\+Mode\+::\+Single}} mode, or \mbox{\hyperlink{classWt_1_1WSelectionBox_a4fcb25c6d7c00f5659959180df9a337c}{set\+Selected\+Indexes()}} and \mbox{\hyperlink{classWt_1_1WSelectionBox_ac54eac01c9ee8b24d27a5de0e02f934c}{selected\+Indexes()}} for Wt\+::\+Selection\+Mode\+::\+Extended mode. The \mbox{\hyperlink{classWt_1_1WComboBox_ac39b46ccd8134a5397b4cbd87dc9f01f}{activated()}} and \mbox{\hyperlink{classWt_1_1WComboBox_a06401499734540c65db26fceb60bfdbf}{sactivated()}} signals are not emited in the Wt\+::\+Selection\+Mode\+::\+Extended mode, use the \mbox{\hyperlink{classWt_1_1WFormWidget_a880589c79f1b93e7f38330e28cf6f5e2}{changed()}} signal.

W\+Selection\+Box is an M\+VC view class, using a simple string list model by default. The model may be populated using \mbox{\hyperlink{classWt_1_1WComboBox_a9b2308a7d962ceb17c1ee32627b106d4}{W\+Combo\+Box\+::add\+Item()}} or \mbox{\hyperlink{classWt_1_1WComboBox_a2092c15c7c0c2f77342d0abdc393ca6b}{W\+Combo\+Box\+::insert\+Item()}} and the contents can be cleared through \mbox{\hyperlink{classWt_1_1WComboBox_a0ed207e9616b51089d97030cc767409d}{clear()}}. These methods manipulate the underlying \mbox{\hyperlink{classWt_1_1WComboBox_ab7ab1df5e7624815fca695243cc7eb0e}{model()}}.

To use the selectionbox with a custom model instead of the default \mbox{\hyperlink{classWt_1_1WStringListModel}{W\+String\+List\+Model}}, use \mbox{\hyperlink{classWt_1_1WComboBox_ac1ae5c539f5b761b4a672ccde09f19f6}{set\+Model()}}.

W\+Selection\+Box is an \mbox{\hyperlink{classWt_1_1WWidget_ac78e3af143883334c82031790c87416e}{inline }} widget.

\doxyparagraph*{C\+SS}

The widget corresponds to the H\+T\+ML {\ttfamily $<$select$>$} tag and does not provide styling. It can be styled using inline or external C\+SS as appropriate.
