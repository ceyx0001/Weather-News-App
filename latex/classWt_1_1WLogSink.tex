\doxysubsection{Detailed Description}
An abstract log sink. 

You can implement this to redirect \mbox{\hyperlink{classWt}{Wt}} or \mbox{\hyperlink{namespaceWt_1_1Dbo}{Wt\+::\+Dbo}}\textquotesingle{}s logging to another logging system.

For \mbox{\hyperlink{classWt}{Wt}}, you can configure the logger on a \mbox{\hyperlink{classWt_1_1WServer}{W\+Server}} level using \mbox{\hyperlink{classWt_1_1WServer_a2f1a0ee0bc7611ee31147808a07ef9e6}{W\+Server\+::set\+Custom\+Logger()}}. For \mbox{\hyperlink{namespaceWt_1_1Dbo}{Wt\+::\+Dbo}}, you can configure the logger globally using \mbox{\hyperlink{namespaceWt_1_1Dbo_a9a326a13084b93cbb33bf1c470879b0d}{Wt\+::\+Dbo\+::set\+Custom\+Logger()}}.
