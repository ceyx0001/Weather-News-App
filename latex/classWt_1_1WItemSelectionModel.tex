\doxysubsection{Detailed Description}
A class that represents a selection for a \mbox{\hyperlink{classWt_1_1WAbstractItemView}{W\+Abstract\+Item\+View}}. 

This model is currently only used by \mbox{\hyperlink{classWt_1_1WTreeView}{W\+Tree\+View}}, and plays only a role in drag \& drop of an item selection.

When an item selection is dragged from a view widget, the generated drop events will have as source object (see \mbox{\hyperlink{classWt_1_1WDropEvent_aa3979c47968c99b3bb81d0d3574acb38}{W\+Drop\+Event\+::source()}}) this selection model.

Although this class does not (yet) allow you to modify the selection, note that manipulations to the model may modify the selection (row insertions and removals may shift the selection, and row deletions may shrink the selection).

\begin{DoxyNote}{Note}
Currently this class cannot be shared between multiple views.
\end{DoxyNote}
