\doxysubsection{Detailed Description}
Exception that indicates a type error. 

This exception is thrown when a \mbox{\hyperlink{classWt_1_1Json_1_1Value}{Value}} is being casted to an incompatible C++ type.

\begin{DoxyNote}{Note}
To avoid exceptions, coerce the type first, and handle Null values. 
\end{DoxyNote}
