\doxysubsection{Detailed Description}
Standard styling for rendering series in charts. 

This class provides four standard palettes, each composed of eight different colors (these are recycled at index 8).

The three colored palettes are a variation on those defined at \href{http://www.modernlifeisrubbish.co.uk/article/web-2.0-colour-palette}{\texttt{ http\+://www.\+modernlifeisrubbish.\+co.\+uk/article/web-\/2.\+0-\/colour-\/palette}}.

The following table lists the background color, and font color of the different palettes\+:

\tabulinesep=1mm
\begin{longtabu}spread 0pt [c]{*{4}{|X[-1]}|}
\hline
{\bfseries{Neutral}} &{\bfseries{Bold}} &{\bfseries{Muted}} &{\bfseries{Gray\+Scale}}  \\\cline{1-4}
Gmail blue &Mozilla red &Ruby on Rails red &Gray \#1  \\\cline{1-4}
Shiny silver &Flock blue &Mozilla blue &Gray \#2  \\\cline{1-4}
Interactive action yellow &R\+SS orange &Etsy vermillion &Gray \#3  \\\cline{1-4}
Qoop mint &Techcrunch green &Digg blue &Gray \#4  \\\cline{1-4}
Digg blue &Flickr pink &43 Things gold &Gray \#5  \\\cline{1-4}
Shadows grey &Newsvine green &Writely olive &Gray \#6  \\\cline{1-4}
Magnolia Mag.\+nolia &Magnolia Mag.\+nolia &Last.\+fm crimson &Gray \#7  \\\cline{1-4}
R\+SS orange &Rollyo red &Basecamp green &Gray \#8  \\\cline{1-4}
\end{longtabu}


The border pen is in all cases a gray pen of 0 width, while the stroke pen is a line of width 2 in the background color.
