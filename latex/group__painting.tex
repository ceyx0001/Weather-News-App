\doxysubsection{Detailed Description}
Classes that provide support for vector graphics painting. 

Wt provides a vector graphics painting system, which depending on the browser support, uses one of four different methods to paint the graphics (inline S\+VG, inline V\+ML, H\+T\+ML 5 $<$canvas$>$ or a raster image). Vector graphics has as benefit a lower bandwidth usage compared to raster images, indepedent of the image size. To use the paint system, you need to specialize \mbox{\hyperlink{classWt_1_1WPaintedWidget}{W\+Painted\+Widget}} and use a \mbox{\hyperlink{classWt_1_1WPainter}{W\+Painter}} to paint the contents of the widget inside its \mbox{\hyperlink{classWt_1_1WPaintedWidget_ad8ce22eff41754c8616f45851f57fb1a}{W\+Painted\+Widget\+::paint\+Event()}}.

In addition, a P\+DF backend is included in the library, which can be used to make a P\+DF version of a painting, or to embed a painting in a P\+DF document.
