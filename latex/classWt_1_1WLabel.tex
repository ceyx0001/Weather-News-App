\doxysubsection{Detailed Description}
A label for a form field. 

The label may contain an image and/or text. It acts like a proxy for giving focus to a \mbox{\hyperlink{classWt_1_1WFormWidget}{W\+Form\+Widget}}. When both an image and text are specified, the image is put to the left of the text.

Usage example\+:

The widget corresponds to the H\+T\+ML {\ttfamily $<$label$>$} tag. When no buddy is set, it is rendered using an H\+T\+ML {\ttfamily $<$span$>$} or {\ttfamily $<$div$>$} to avoid click event handling misbehavior on Microsoft Internet Explorer.

W\+Label is an \mbox{\hyperlink{classWt_1_1WWidget_ac78e3af143883334c82031790c87416e}{inline }} widget.

\doxyparagraph*{C\+SS}

This widget does not provide styling, and can be styled using inline or external C\+SS as appropriate. A label\textquotesingle{}s text may be styled via a nested {\ttfamily $<$span$>$} element, and it\textquotesingle{}s image may be styled via a nested {\ttfamily $<$img$>$} element.

Sets the text format. 

The text\+Format controls how the string should be interpreted\+: either as plain text, which is displayed literally, or as X\+H\+T\+M\+L-\/markup.

When changing the text\+Format to Wt\+::\+Text\+Format\+::\+X\+H\+T\+ML, and the current text is literal (not created using \mbox{\hyperlink{classWt_1_1WString_adf1e7e5fc3c7199563a69bf07d77e571}{W\+String\+::tr()}}), the current text is parsed using an X\+ML parser which discards malicious tags and attributes silently. When the parser encounters an X\+ML parse error, the text\+Format is left unchanged, and this method returns false.

Returns whether the text\+Format could be set for the current text.

The default format is Wt\+::\+Text\+Format\+::\+X\+H\+T\+ML. \mbox{\Hypertarget{classWt_1_1WLabel_a50e413fa171f301129b39a4d4ed6d2fd}\label{classWt_1_1WLabel_a50e413fa171f301129b39a4d4ed6d2fd}}

Configures word wrapping. 

When {\ttfamily word\+Wrap} is {\ttfamily true}, the widget may break lines, creating a multi-\/line text. When {\ttfamily word\+Wrap} is {\ttfamily false}, the text will displayed on a single line, unless the text contains end-\/of-\/lines (for Wt\+::\+Text\+Format\+::\+Plain) or $<$br /$>$ tags or other block-\/level tags (for Wt\+::\+Text\+Format\+::\+X\+H\+T\+ML).

The default value is {\ttfamily false}.
