Processes the (initial) environment. 

This method process environmental information that may be relevant to authentication\+:


\begin{DoxyItemize}
\item email tokens, which are indicated through an internal path. The widget uses dialogs (by default) to interact with the user to act on the token.
\item authentication tokens, which are stored in browser cookies, to implement remember-\/me functionality. When logging in using an authentication token, the login is considered \char`\"{}weak\char`\"{} (since a user may have inadvertently forgotten to logout from a public computer). You should let the user authenticate using another, primary method before doing sensitive operations. The \mbox{\hyperlink{classWt_1_1Auth_1_1AuthWidget_a5f933e20250589dbdfd263b523e43d7d}{create\+Password\+Prompt\+Dialog()}} method may be useful for this.
\end{DoxyItemize}
