\doxysubsection{Detailed Description}
A clickable option. 

This widget is part of the Wt composer example.

On its own, an option is a text which is style \char`\"{}option\char`\"{}. An \mbox{\hyperlink{classOption}{Option}} may also be used as items in an \mbox{\hyperlink{classOptionList}{Option\+List}}.
