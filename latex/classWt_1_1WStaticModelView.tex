\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Renderer$>$\newline
class Wt\+::\+W\+Static\+Model\+View$<$ Renderer $>$}

A widget that implements a view for a non-\/changing model. 

This model uses a function object which is passed in the constructor to render the View, and does not react to changes.

You may want to use the utility function Wt\+::make\+Static\+Model() to create an instance of this class.
