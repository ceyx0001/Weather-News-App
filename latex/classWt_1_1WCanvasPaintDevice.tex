\doxysubsection{Detailed Description}
A paint device for rendering using the H\+T\+ML 5 $<$canvas$>$ element. 

The W\+Canvas\+Paint\+Device is used by \mbox{\hyperlink{classWt_1_1WPaintedWidget}{W\+Painted\+Widget}} to render to the browser using the H\+T\+ML 5 $<$canvas$>$ element. You usually will not use the device directly, but rather rely on \mbox{\hyperlink{classWt_1_1WPaintedWidget}{W\+Painted\+Widget}} to use this device when appropriate.

\begin{DoxyNote}{Note}
Older browsers do not have text support in $<$canvas$>$. \mbox{\hyperlink{structText}{Text}} is then rendered in an overlayed D\+IV and a consequence text is not subject to rotation and scaling components of the current transformation (but does take into account translation). On most browser you can use the \mbox{\hyperlink{classWt_1_1WSvgImage}{W\+Svg\+Image}} or \mbox{\hyperlink{classWt_1_1WVmlImage}{W\+Vml\+Image}} paint devices which do support text natively. 
\end{DoxyNote}

Draws an arc. 

The arc describes the segment of an ellipse enclosed by the rect. The segment starts at {\ttfamily start\+Angle}, and spans an angle given by {\ttfamily span\+Angle}. These angles have as unit degree, and are measured counter-\/clockwise starting from the 3 o\textquotesingle{}clock position.

The arc must be stroked, filled, and transformed using the current painter settings.
