\doxysubsection{Detailed Description}
A validator that checks the string length of user input. 

This validator checks whether user input is within the specified range of accepted string lengths.

If you only want to limit the length on a line edit, you may also use \mbox{\hyperlink{classWt_1_1WLineEdit_aaeca334d8a47be029b6a44912b01ed5d}{W\+Line\+Edit\+::set\+Max\+Length()}}.

\doxyparagraph*{i18n}

The strings used in this class can be translated by overriding the default values for the following localization keys\+:
\begin{DoxyItemize}
\item Wt.\+W\+Length\+Validator.\+Too\+Short\+: The input must be at least \{1\} characters
\item Wt.\+W\+Length\+Validator.\+Bad\+Range\+: The input must have a length between \{1\} and \{2\} characters
\item Wt.\+W\+Length\+Validator.\+Too\+Long\+: The input must be no more than \{1\} characters 
\end{DoxyItemize}
