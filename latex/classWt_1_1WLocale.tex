\doxysubsection{Detailed Description}
A locale. 

This class provides localization support for an application.

It can be used to determine the formatting of dates and numbers, and by \mbox{\hyperlink{classWt_1_1WLocalDateTime}{W\+Local\+Date\+Time}} to determine the time zone.

\mbox{\hyperlink{classNote}{Note}} that only the \mbox{\hyperlink{classWt_1_1WLocale_aa0d986a53d1fac375e2c614c392d01be}{name()}} parameter is automatically retrieved from the browser, if possible, and message resources are deduced based on that. All other parameters have to be configured.

Sets the time zone. 

This sets the time zone (used by the client). This is a time zone from Howard Hinnant\textquotesingle{}s date library, which will become integrated into the C++20 standard.

A time zone can be retrieved with {\ttfamily date\+::locate\+\_\+zone} using the name of the time zone. The time zone name may be retrieved with \mbox{\hyperlink{classWt_1_1WEnvironment_ab429100601e457420dba6c436b97ba20}{W\+Environment\+::time\+Zone\+Name()}}, if supported by the browser.

\mbox{\hyperlink{classWt_1_1WEnvironment_abe9ace6cd3b5c61e2d20cea5ef493e6c}{W\+Environment\+::time\+Zone\+Offset()}} will also (if available) contain an offset. This offset could be used to let the user select a time zone from a list of timezones. You can get a list of zones with {\ttfamily date\+::get\+\_\+tzdb()}.zones.

The timezone is used by \mbox{\hyperlink{classWt_1_1WLocalDateTime}{W\+Local\+Date\+Time}}.
