\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Type$>$\newline
struct Wt\+::any\+\_\+traits$<$ Type $>$}

A traits class for a type stored in a cpp17\+::any. 

The traits are used by Wt\textquotesingle{}s view classes (such as \mbox{\hyperlink{classWt_1_1WTableView}{Wt\+::\+W\+Table\+View}} \mbox{\hyperlink{classWt_1_1WTreeView}{Wt\+::\+W\+Tree\+View}}, \mbox{\hyperlink{classWt_1_1Chart_1_1WCartesianChart}{Wt\+::\+Chart\+::\+W\+Cartesian\+Chart}}) to deal with values in a \mbox{\hyperlink{classWt_1_1WAbstractItemModel}{Wt\+::\+W\+Abstract\+Item\+Model}}\+:
\begin{DoxyItemize}
\item conversion to a string by \mbox{\hyperlink{group__modelview_ga2f96ad66a5212499e5b4d08609800a64}{Wt\+::as\+String()}}
\item conversion to a number by \mbox{\hyperlink{group__modelview_ga48a507458cc9f4754fe8b6459f4ad3d1}{Wt\+::as\+Number()}}
\item sorting by \mbox{\hyperlink{classWt_1_1WSortFilterProxyModel}{Wt\+::\+W\+Sort\+Filter\+Proxy\+Model}} or \mbox{\hyperlink{classWt_1_1WStandardItemModel}{W\+Standard\+Item\+Model}}.
\end{DoxyItemize}

This traits class provides a default implementation for supporting values {\itshape value} of a custom type {\itshape Type} in Wt\textquotesingle{}s M\+VC system, which relies on\+:
\begin{DoxyItemize}
\item serializing to a stream (std\+::ostream $<$$<$ {\itshape value}) for converting the value to a string.
\item lexical interpretation as a number for converting the value to a number, using std\+::stod()
\item sorting using operator== and operator$<$
\end{DoxyItemize}

You can specialize these traits if you need to customize one or more of these operations.
