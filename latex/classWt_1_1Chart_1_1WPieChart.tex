\doxysubsection{Detailed Description}
A pie chart. 

A pie chart renders a single data series as segments of a circle, so that the area of each segment is proportional to the value in the data series.

To use a pie chart, you need to set a model using \mbox{\hyperlink{classWt_1_1Chart_1_1WAbstractChart_a2243e89c5ee56f48da4b1ca49e854c0f}{set\+Model()}}, and use \mbox{\hyperlink{classWt_1_1Chart_1_1WPieChart_aa40bfd3f2095106f8081596b235de4c6}{set\+Labels\+Column()}} and \mbox{\hyperlink{classWt_1_1Chart_1_1WPieChart_a3692ea22a1417e599e9358b0ed5a723d}{set\+Data\+Column()}} to specify the model column that contains the category labels and data.

The pie chart may be customized visually by enabling a 3D effect (\mbox{\hyperlink{classWt_1_1Chart_1_1WPieChart_ae36d57f4c2801cb8599a1bc99e340a24}{set\+Perspective\+Enabled()}}), or by specifying the angle of the first segment. One or more segments may be exploded, which separates the segment from the rest of the pie chart, using \mbox{\hyperlink{classWt_1_1Chart_1_1WPieChart_a4a2bfe0c147b5433d340d49351b5032c}{set\+Explode()}}.

The segments may be labeled in various ways using \mbox{\hyperlink{classWt_1_1Chart_1_1WPieChart_a89324f6159e5893e1ddd368718d00e5d}{set\+Display\+Labels()}}.
