\doxysubsection{Detailed Description}
Abstract delegate class for creating form widgets. 

Form delegates will be automatically generated using \mbox{\hyperlink{namespaceWt_1_1Dbo}{Wt\+::\+Dbo}}\textquotesingle{}s persist method. Depending on the data type used by the Dbo class, the form delegate will have a different behavior. For example\+: for \mbox{\hyperlink{classWt_1_1WString}{W\+String}} objects the default widget in the view will be a \mbox{\hyperlink{classWt_1_1WLineEdit}{W\+Line\+Edit}}.

Form delegates will be automatically generated when calling \mbox{\hyperlink{classWt_1_1Form_1_1Dbo_1_1FormView_ade0ef447f3d5711eb4819d8da883ef3b}{Dbo\+::\+Form\+View$<$\+C$>$\+::set\+Form\+Model()}}. This Form\+View also has the option to override the delegates that are generated by using \mbox{\hyperlink{classWt_1_1Form_1_1Dbo_1_1FormView_a42cad48a1c44d319325be1732520ba83}{Dbo\+::\+Form\+View$<$\+C$>$\+::set\+Form\+Delegate()}}.

Form delegates will also create the corresponding \mbox{\hyperlink{classWt_1_1WValidator}{W\+Validator}} and update the \mbox{\hyperlink{classWt_1_1WFormModel}{W\+Form\+Model}} and \mbox{\hyperlink{classWt_1_1WTemplateFormView}{W\+Template\+Form\+View}} accordingly.
