\doxysubsection{Detailed Description}
A widget that provides a drop-\/down combo-\/box control. 

A combo box provides the user with a set of options, from which one option may be selected.

W\+Combo\+Box is an M\+VC view class, using a simple string list model by default. The model may be populated using \mbox{\hyperlink{classWt_1_1WComboBox_a9b2308a7d962ceb17c1ee32627b106d4}{add\+Item(const W\+String\&)}} or \mbox{\hyperlink{classWt_1_1WComboBox_a2092c15c7c0c2f77342d0abdc393ca6b}{insert\+Item(int, const W\+String\&)}} and the contents can be cleared through \mbox{\hyperlink{classWt_1_1WComboBox_a0ed207e9616b51089d97030cc767409d}{clear()}}. These methods manipulate the underlying \mbox{\hyperlink{classWt_1_1WComboBox_ab7ab1df5e7624815fca695243cc7eb0e}{model()}}.

To use the combo box with a custom model instead of the default \mbox{\hyperlink{classWt_1_1WStringListModel}{W\+String\+List\+Model}}, use \mbox{\hyperlink{classWt_1_1WComboBox_ac1ae5c539f5b761b4a672ccde09f19f6}{set\+Model()}}.

To react to selection events, connect to the \mbox{\hyperlink{classWt_1_1WFormWidget_a880589c79f1b93e7f38330e28cf6f5e2}{changed()}}, \mbox{\hyperlink{classWt_1_1WComboBox_ac39b46ccd8134a5397b4cbd87dc9f01f}{activated()}} or \mbox{\hyperlink{classWt_1_1WComboBox_a06401499734540c65db26fceb60bfdbf}{sactivated()}} signals.

At all times, the current selection index is available through \mbox{\hyperlink{classWt_1_1WComboBox_a46d536faa379cf01e50974065265780f}{current\+Index()}} and the current selection text using \mbox{\hyperlink{classWt_1_1WComboBox_a815e7bf650f01348cae2a2594c7c4cab}{current\+Text()}}.

\mbox{\hyperlink{classWt_1_1WComboBox}{W\+Combo\+Box}} does not have support for auto-\/completion, this behaviour can be found in the \mbox{\hyperlink{classWt_1_1WSuggestionPopup}{W\+Suggestion\+Popup}}.

W\+Combo\+Box is an \mbox{\hyperlink{classWt_1_1WWidget_ac78e3af143883334c82031790c87416e}{inline }} widget.

\doxyparagraph*{C\+SS}

The widget corresponds to the H\+T\+ML {\ttfamily $<$select$>$} tag and does not provide styling. It can be styled using inline or external C\+SS as appropriate.
