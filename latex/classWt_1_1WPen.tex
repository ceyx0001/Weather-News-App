\doxysubsection{Detailed Description}
A value class that defines the style for pen strokes. 

A pen defines the properties of how lines (that may surround shapes) are rendered.

A pen with width 0 is a {\itshape cosmetic} pen, and is always rendered as 1 pixel width, regardless of transformations. Otherwized, the pen width is modified by the \mbox{\hyperlink{classWt_1_1WPainter_a0a509496b00579ec993d91850c44e05d}{transformation}} set on the painter.

\doxyparagraph*{Java\+Script exposability}

A W\+Pen is Java\+Script exposable. If a W\+Pen \mbox{\hyperlink{classWt_1_1WJavaScriptExposableObject_a4e121ccbb14715c33dc1b019b6636156}{is Java\+Script bound}}, it can be accessed in your custom Java\+Script code through \mbox{\hyperlink{classWt_1_1WJavaScriptHandle_a0d0bc00f0bcafb596c3f6c96c4241ded}{its handle\textquotesingle{}s js\+Ref()}}. At the moment, only the \mbox{\hyperlink{classWt_1_1WPen_a580615277eacb0bd1c6796d04c09bb09}{color()}} property is exposed, e.\+g. a pen with the color W\+Color(10,20,30,255) will be represented in Java\+Script as\+:

\DoxyCodeLine{  \mbox{\hyperlink{classWt_1_1WPen_a580615277eacb0bd1c6796d04c09bb09}{color}}: [10,20,30,255]}
\DoxyCodeLine{\}}
\end{DoxyCode}


\begin{DoxyWarning}{Warning}
