Refresh the widget. 

The refresh method is invoked when the locale is changed using \mbox{\hyperlink{classWt_1_1WApplication_ac0af7e0131c926c77b207595c8c6e283}{W\+Application\+::set\+Locale()}} or when the user hit the refresh button.

The widget must actualize its contents in response.

\begin{DoxyNote}{Note}
This does {\bfseries{not}} rerender the widget! Calling \mbox{\hyperlink{classWt_1_1SoundManager_a341e89cf31da83cb862d9e4e4821543e}{refresh()}} usually does not have any effect (unless you\textquotesingle{}ve reimplemented \mbox{\hyperlink{classWt_1_1SoundManager_a341e89cf31da83cb862d9e4e4821543e}{refresh()}} to attach to it an effect). 
\end{DoxyNote}
