\doxysubsection{Detailed Description}
A widget that displays a google map. 

This widget uses the online Google Maps server to display a map. It exposes a part of the google maps A\+PI.

This widget supports both version 2 and version 3 of the Google Maps A\+PI. The version 2 A\+PI is used by default, to enable the version 3 A\+PI, use the constructor\textquotesingle{}s version argument.

To use the map on a public server you will need to obtain a key. The widget will look for this key as the configuration property {\ttfamily \char`\"{}google\+\_\+api\+\_\+key\char`\"{}}. If this configuration property has not been set, it will use a key that is suitable for localhost.

\doxyparagraph*{C\+SS}

Styling through C\+SS is not applicable.
